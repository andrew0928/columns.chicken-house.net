# 總算搬完了

## 问题与答案 (FAQ)

### Q&A 類別 A: 概念理解類

Q1: 什麼是家用伺服器與桌機的主機板對調？
- A簡: 將兩台電腦的主機板（含CPU、散熱、記憶體與連接線）互換，並完成驅動、啟用與穩定性檢查的作業流程。
- A詳: 主機板對調是把A機的主機板與CPU等關鍵元件移至B機，並把B機對應零件移至A機。動機多為性能、功耗與噪音最佳化，例如把更省電安靜的Core 2 Duo放進長開機的家用伺服器，而把Pentium D挪至娛樂桌機。特點是牽涉硬體相容、散熱空間、線材與前置面板接線，且系統可能要求Windows重新啟用。應用場景包括家庭影音中心（MCE）、家庭雲與輕量服務器升級。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q1, A-Q3, A-Q4, C-Q1

Q2: 什麼是 Windows Media Center Edition（MCE）？
- A簡: 以Windows XP為基礎的媒體中心版本，整合電視錄影、影音播放、遙控與10呎介面，適合客廳與HTPC。
- A詳: Windows XP MCE是微軟為居家影音設計的系統，提供10呎介面、遙控支援、電視節目導覽與錄影（需TV Tuner），並整合相片、音樂、影片播放。特點是對硬體相容與驅動要求較高，常依賴顯示卡的DXVA硬體加速以降低CPU負載。應用場景是HTPC、家庭錄影機、客廳多媒體播放，與安靜、低功耗硬體搭配可獲得更佳體驗。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q10, B-Q14, C-Q3, D-Q8

Q3: 為什麼把 Core 2 Duo 換到 server 上？
- A簡: 因Core 2 Duo具高效能/功耗比與低噪音，對長時間運作的家用伺服器更合適。
- A詳: 家用伺服器常年運作，功耗與噪音是關鍵。Core 2 Duo相較Pentium D，在相近工作負載下更省電、更涼，因此能用較低風扇轉速獲得安靜運行，同時提升處理效能（如檔案服務、轉檔、備份）。把C2D換到伺服器，讓長開機設備更高效穩定；把Pentium D移到以播放為主的桌機，也可藉由顯卡解碼降低CPU負載，滿足娛樂需求。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q4, B-Q3, C-Q1

Q4: Pentium D 920 與 Core 2 Duo 的差異？
- A簡: C2D效能/功耗比更佳、發熱更低、噪音可控；Pentium D發熱大、效率低，適合次要或依賴硬體解碼的用途。
- A詳: Pentium D 920採NetBurst雙核心架構，時脈高但每時鐘效能低、TDP高，散熱與噪音壓力大。Core 2 Duo（Conroe）具更短管線與高IPC，動態電壓/頻率管理較佳，TDP較低。在同級負載下，C2D提供更高效能與更少熱量。應用上，C2D適合長時運作的伺服器或需要即時編解碼的多工；Pentium D可退居影音播放（依DXVA減輕CPU）或次要工作站。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q3, A-Q12, D-Q10

Q5: 為什麼硬體升級會觸發 Windows XP 重新啟用？
- A簡: XP以硬體指紋比對重要元件變化。主機板等大幅更動觸發驗證，需要線上或電話啟用。
- A詳: Windows XP使用「硬體雜湊」比較系統元件（主機板、CPU、網卡、磁碟控制器等）。當變更分數超過門檻（例如更換主機板/CPU組），系統判定為新硬體環境，要求重新啟用授權。啟用流程包含網路自動啟用或電話啟用（輸入安裝ID與取得確認ID）。應用場景：更換主機板、將系統移植到另一台機器、重大升級後恢復合法使用。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q2, C-Q9, D-Q5

Q6: 什麼是前置面板接線？有哪些常見錯誤？
- A簡: 連接電源鍵、重置鍵、指示燈與前置USB/音訊的細排線；常見接反極性或插錯針腳。
- A詳: 機殼前置面板包含Power SW、Reset SW、Power LED、HDD LED、前置USB/音訊等，需依主機板針腳圖連接。指示燈有極性，接反不亮；按鍵無極性；USB 9-pin排針若插錯，可能導致無法使用或危及裝置。常見錯誤是未閱讀主機板手冊、線材未標示正負、LED與USB混插。應用場景：裝機、對調後復線、排除「燈不亮、USB不能用」。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q5, C-Q4, D-Q6

Q7: USB 鍵盤開機前抓不到的常見原因？
- A簡: BIOS未啟用Legacy USB、特定KVM不相容、供電不足或韌體舊。可啟用Legacy或用PS/2。
- A詳: 在作業系統載入前，USB鍵盤需由BIOS透過Legacy USB模擬PS/2輸入。若該選項關閉、韌體過舊、或KVM/延長線不相容，會導致於啟動畫面無法輸入。症狀是無法按F1/Del或完成啟用流程。解法：以PS/2鍵盤進入BIOS，啟用Legacy USB；更新BIOS；改用直連或供電KVM。應用：開機設定、XP啟用確認、RAID控制器設定。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q4, C-Q2, D-Q7

Q8: 什麼是具鎖扣的 SATA 線？為何推薦？
- A簡: 具有金屬扣鎖的SATA線，能防止接頭鬆脫，提高抗震與可靠度，適合頻繁拆裝或擁擠機殼。
- A詳: 早期SATA線為摩擦固定，易於拉扯或震動中鬆脫，導致隨機斷線、磁碟離線。具鎖扣的SATA線在接頭加入金屬彈片卡扣，插入對應鎖槽後更牢靠。優點是抗拔插與震動、改善線材管理；注意需主機板/硬碟端支援鎖槽。應用：緊湊機殼、行動/側放機殼、頻繁維護、確保錄影不中斷。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q6, C-Q5, D-Q4

Q9: 顯示卡散熱器與機壳風扇干涉是什麼？
- A簡: 顯卡長度/厚度與機殼內風扇位置衝突，導致無法闔側板或裝入，需調整配置或選件。
- A詳: 干涉多由機殼內部空間不足、顯卡散熱器過厚/過長、或側板/風扇位置過近造成。表現為側板蓋不上、風扇碰觸散熱片或線材擠壓。解法包含改用薄型風扇、移位風扇、改選短板/單槽顯卡、移除不必要的硬碟籠、或更換相容機殼。應用：HTPC小殼、雙槽顯卡安裝、安靜風扇佈局。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q7, C-Q6, D-Q1

Q10: 大尺寸機殼風扇的優缺點？
- A簡: 低轉速大風量、噪音低；但容易吸塵/昆蟲、占空間、可能與零件干涉，需濾網與良好佈局。
- A詳: 大尺寸風扇（如120mm以上）能以較低轉速提供足夠風量，降低噪音與渦流；適合安靜PC。但其開口面積大，若未加濾網，容易吸入灰塵與小昆蟲；厚度與框體也可能干涉長顯卡或側板。最佳實踐是加裝易清理濾網、規劃正壓/微正壓進氣、避開干涉區、並以PWM控制平衡溫度與噪音。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q8, B-Q18, C-Q7, D-Q2

Q11: 為何需要進行灰塵與進氣管理？
- A簡: 灰塵阻礙散熱、增噪與故障；良好進氣與濾網能維持溫度穩定、延長壽命並減少維護。
- A詳: 灰塵會覆蓋散熱鰭片與濾網、降低熱對流效率，迫使風扇加速而增噪；亦可能導致軸承磨損、電容過熱、甚至短路。進氣管理透過濾網、正壓氣流與定期清潔，減少塵埃進入與積聚。適當的線材管理也能改善風道、降低風阻。應用於長時間運作的伺服器、HTPC與粉塵環境中的個人電腦。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q9, C-Q7, D-Q2

Q12: 安靜電腦（Quiet PC）的核心價值是什麼？
- A簡: 以低噪音與穩定為目標，兼顧散熱、功耗與性能，提升居家/辦公舒適與可靠度。
- A詳: 安靜電腦注重在可用性能下把噪音降至不干擾的水準，透過選擇低TDP處理器、優質散熱器、大風扇低轉速、阻尼與防震、優化風道與風扇曲線等達成。優點是長時間使用不疲勞、錄音/客廳環境友善，且較低溫度增進穩定性。典型場景是HTPC、家用伺服器與工作站夜間運作。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q10, B-Q11, C-Q8, D-Q10

Q13: 什麼是 BIOS 的 Legacy USB 支援？
- A簡: 一種在作業系統前，將USB鍵鼠模擬為PS/2輸入的韌體功能，便於進入BIOS與安裝。
- A詳: Legacy USB是主機板韌體（BIOS/UEFI）提供的相容層，透過SMM或中斷服務將USB HID事件轉譯為傳統控制器事件。若關閉，開機前USB鍵鼠可能無作用，影響設定、RAID、啟用流程。建議在需要前開啟，待OS安裝好驅動後可視需求調整。對KVM與延長線相容性也有助益。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q4, C-Q2, D-Q7

Q14: MCE 與一般 Windows XP 的差別？
- A簡: MCE加上10呎介面、電視錄影與遙控支援，需相容TV卡與顯示卡驅動；XP無此整合。
- A詳: MCE在XP基礎上整合Media Center外殼、EPG、錄影排程與遙控器介面，對影音硬體加速（DXVA）與BDA驅動依賴較高。一般XP僅提供多媒體基礎功能，需第三方軟體整合。MCE適合HTPC與客廳環境；XP更通用。重大硬體變更後，MCE可能需要重新偵測裝置、安裝驅動與啟用。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q10, B-Q14, C-Q3, D-Q8

Q15: 為何安排維護時段很重要？
- A簡: 降低對家人與服務的中斷，降低壓力並預留回復時間；適合在非尖峰或家人外出時進行。
- A詳: 硬體對調涉及停機與反覆測試。選在家人外出或非錄影/備份尖峰時段施工，可避免服務中斷（如MCE錄影遺漏）、降低趕工錯誤率，並保留足夠時間處理突發問題（如啟用、驅動、干涉）。預先規劃時程與回復方案（回滾、備機）可顯著提高成功率。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q1, C-Q10, D-Q8

Q16: 什麼是硬體/軟體驗證清單？
- A簡: 對調後逐項檢查的測試列表，涵蓋開機、溫度、I/O、網路、影音與備份等，確保穩定運作。
- A詳: 驗證清單將風險點具體化，如：POST/BIOS、磁碟與SATA穩定、風扇轉速與溫度、前置USB/讀卡機、網路吞吐、顯示與音效、MCE錄影與回放、待機/喚醒、SMART/記憶體測試等。依序執行並記錄結果，搭配壓力測試（如Prime95、FurMark、CrystalDiskMark、MemTest）與事件檢視器，能及早發現問題。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q10, D-Q9, B-Q13

### Q&A 類別 B: 技術原理類

Q1: 主機板對調的標準流程與關鍵步驟？
- A簡: 備份與關機放電、標記線材、拆卸/清潔、移植CPU/散熱與記憶體、重新佈線、首次開機測試與驅動。
- A詳: 原理是以模組化拆裝降低風險。關鍵步驟：1) 備份資料、建立還原點；2) 斷電、長按電源釋放餘電；3) 拍照與標記前置面板、SATA、電源線；4) 拆主機板與CPU/散熱器，清潔灰塵並重上導熱膏；5) 安裝到另一機殼、檢查支撐柱位置；6) 接上ATX/EPS、SATA、前置面板與風扇；7) 首次開機進BIOS載入預設、檢查溫度與儲存模式；8) 進OS安裝驅動、完成啟用；9) 壓力測試與驗證清單。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q1, C-Q1, C-Q10

Q2: Windows XP 啟用如何計算硬體變更？
- A簡: 以硬體雜湊（多項元件權重）比對，新雜湊與舊值差異超門檻即要求啟用。
- A詳: XP針對主機板晶片組、CPU、RAM數量、HDD序號、網卡MAC、光碟機等建立雜湊權重。更換多項關鍵（尤其是主機板/CPU/網卡）會超過允差。首次啟用後有鎖定期與次數限制。流程：蒐集硬體資訊→產生安裝ID→伺服器比對→回傳確認ID。核心組件是授權服務、WPA子系統與硬體檢測模組。建議在重大更換前備妥授權資訊。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q5, C-Q9, D-Q5

Q3: Core 2 Duo 相對於 Pentium D 的功耗降低機制？
- A簡: 更高IPC、短管線、智慧電源管理與更先進製程，使同效能下功耗更低、溫度更小。
- A詳: C2D（Conroe）相較NetBurst（Pentium D），採更有效的微架構：更短管線減少錯誤預測損耗、更大快取與共享架構提升命中、Wide Issue提升IPC。配合先進製程與SpeedStep動態電壓/頻率調整、閒置關閉未用單元，功耗顯著下降。關鍵組件：電源管理單元、L2快取、前端預取。結果是同等工作下風扇可低轉，噪音與熱壓力降低。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q4, A-Q12, D-Q10

Q4: BIOS 如何在無驅動下支援 USB 鍵盤？
- A簡: 透過Legacy USB在SMM中攔截USB HID事件，模擬PS/2中斷，供POST/Setup使用。
- A詳: BIOS在SMM（System Management Mode）執行USB控制邏輯，將USB鍵盤/滑鼠的HID報文轉譯為傳統8042控制器訊號。流程：USB枚舉→HID報文→SMM轉換→系統回應，核心組件是USB控制器、SMM代碼、8042模擬。若關閉或韌體不佳，開機時鍵盤無效。可於BIOS啟用「Legacy USB Support」，或使用原生PS/2裝置。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: A-Q7, C-Q2, D-Q7

Q5: 前置面板接線的電氣與腳位機制？
- A簡: LED為有極性二極體，按鍵為短路開關；USB 9-pin依標準供電/資料腳位排列。
- A詳: LED線為二極體，正極接PLED+/HLED+，反向不亮；Power/Reset為瞬時短接開關。USB 2.0前置針腳常見9-pin（5+4）排列：VCC、D-、D+、GND，缺角定位；插反可能短路或裝置失效。音訊前置有HD Audio/AC’97兩種針腳定義。核心組件：主機板前置排針、插頭標示。務必參照主機板手冊對應連接。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q6, C-Q4, D-Q6

Q6: SATA 接頭易鬆脫的機械原因與改良？
- A簡: 早期摩擦式無鎖扣，受震動/拉扯易脫落；改用有鎖扣線與理線固定可改善。
- A詳: SATA 7-pin連接器早期依靠摩擦力與塑膠摩擦面固定，受機殼震動、線材張力或維修拉扯易鬆脫。改良方式是使用帶金屬鎖扣的SATA線配合帶鎖槽的插座。流程：選對鎖扣規格→按壓卡住→以紮帶固定應力釋放點。核心組件：SATA連接器、鎖扣彈片、理線夾具。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q8, C-Q5, D-Q4

Q7: 顯卡與機殼相容性的規範與容差？
- A簡: 依ATX/PCIe規範保留外形與禁放區，但散熱器厚度/長度差異大，需實測空間。
- A詳: ATX/PCIe規範定義主機板插槽位置與卡邊界，但未強制散熱器外形高度/長度，實務上雙槽、長板與背板高度各異。機殼亦有硬碟籠、風扇位、側板限高。流程：量測機殼顯卡空間→比對顯卡尺寸→檢查風扇/線材干涉。核心組件：PCIe插槽、I/O檔板、機殼立柱與風扇位。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q9, C-Q6, D-Q1

Q8: 機殼氣流設計如何影響溫度與噪音？
- A簡: 正確的進/出風路徑與風量平衡可降低溫度與風扇轉速，達到更低噪音與穩定。
- A詳: 氣流原理是冷空氣由前/下進、熱空氣由後/上出，形成定向風道。正壓可減少灰塵進入縫隙，微正壓常見於安靜機殼。風扇數量與轉速需平衡，避免亂流。流程：規劃進出風→選扇→設定曲線→測溫。核心組件：進氣濾網、風扇（PWM/DC）、風道屏障（擋板/線材）。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q10, A-Q11, C-Q8

Q9: 灰塵如何削弱散熱與增加故障率？
- A簡: 灰塵堵塞鰭片與濾網、降低對流，迫使風扇加速增噪，並可能致短路與軸承磨損。
- A詳: 灰塵形成熱障，讓散熱鰭片與濾網通風受阻，風扇為維持風量需提高轉速與噪音。積塵也會使風扇失衡振動、靜電吸附於電路造成漏電風險。流程：定期清潔→更換或清洗濾網→檢查風扇與溫度曲線。核心組件：散熱器、濾網、風扇軸承、電源供應器進氣口。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q11, C-Q7, D-Q2

Q10: MCE 的錄影播放如何利用硬體加速？
- A簡: 藉顯卡DXVA解碼MPEG-2/WMV，減輕CPU負載；TV Tuner以BDA驅動串接濾鏡。
- A詳: MCE透過DirectShow/EVR與DXVA調用顯示卡的影片解碼與後處理（去交錯、縮放），降低CPU占用。錄影透過BDA驅動構建濾鏡圖：Tuner→Demod→Capture→Encoder→Mux→File Writer。核心組件：GPU驅動、解碼器、BDA驅動、系統時鐘同步。正確驅動與硬體加速設定可提升播放與錄影穩定度。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q2, A-Q14, C-Q3, D-Q8

Q11: 風扇轉速控制（PWM/DC）的原理與差異？
- A簡: PWM以占空比控制4-pin風扇轉速；DC以電壓調整3-pin風扇轉速，前者效率與範圍更好。
- A詳: PWM使用第四針腳輸入調速訊號（25kHz常見），在固定12V供電下改變占空比控制馬達轉速；DC調速則透過降低電壓（如7V、9V）減速，低壓可能啟動不穩。流程：BIOS/軟體設定曲線→溫度感測→控制器輸出。核心組件：主機板風扇控制器、溫度感測器、PWM/DC風扇。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q12, C-Q8, D-Q10

Q12: 主機板更換時儲存控制模式（AHCI/IDE）如何影響開機？
- A簡: 模式不符驅動將導致0x7B無法開機；需先在原系統啟用驅動或改回相容模式。
- A詳: Windows對AHCI/IDE模式需對應驅動。若原系統以IDE裝置安裝、更換主機板後改為AHCI，會因無驅動在啟動期載入而藍屏（0x7B）。流程：更換前啟用AHCI驅動（登錄/服務）、或更換後先以IDE開機再安裝驅動後切換。核心組件：StorAHCI/廠商F6驅動、BIOS儲存設定。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q1, C-Q10, D-Q9

Q13: 開機自我測試（POST）與 beep code 代表什麼？
- A簡: POST檢查CPU、RAM、顯示輸出與基本I/O；蜂鳴碼對應故障元件，利於無畫面診斷。
- A詳: 開機加電後，BIOS執行POST：初始化CPU、測試RAM、偵測顯卡與裝置。若出錯，透過蜂鳴器以長短聲組合提示（依主機板/BIOS廠牌）。流程：加電→自檢→輸出→交給bootloader。核心組件：BIOS韌體、南北橋/晶片組、蜂鳴器。掌握beep code有助快速定位無法開機問題。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q10, D-Q9, A-Q16

Q14: TV Tuner 在 MCE 中的 BDA 架構如何運作？
- A簡: 以BDA驅動暴露統一介面，透過DirectShow建立濾鏡圖完成調諧、解調、擷取與封裝。
- A詳: BDA（Broadcast Driver Architecture）定義TV裝置的標準驅動與濾鏡介面。MCE呼叫BDA元件建立Graph：Tuner設定頻道→Demod解調→Capture擷取TS→Encoder（類比時）壓縮→MPG/MXF輸出。流程：驅動載入→裝置枚舉→Graph連線→同步錄製。核心組件：BDA驅動、DirectShow、硬體Tuner/Demod。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: B-Q10, C-Q3, D-Q8

### Q&A 類別 C: 實作應用類

Q1: 如何規劃並安全地對調兩台電腦的主機板與 CPU？

- A詳: 步驟：1) 完整備份、記錄授權金鑰與驅動；2) 規劃停機時段；3) 斷電、去靜電；4) 拍照標記前置面板/SATA/風扇線；5) 拆下散熱器、清潔與重新塗抹導熱膏；6) 按主機板孔位調整支撐柱、固定主機板；7) 連接ATX/EPS、SATA（優先鎖扣線）、前置I/O；8) 首開進BIOS載入Optimized Defaults，確認溫度與儲存模式；9) 進OS安裝晶片組/網卡/音效驅動，完成啟用；10) 執行MemTest、溫度與燒機。注意：避免過度上膏、線材拉力，螺絲對角緊固。

Q2: 如何在 BIOS 啟用 Legacy USB 讓鍵盤可用？

Q3: 如何為 MCE 設定錄影品質與硬體加速？

- A詳: 步驟：1) 安裝GPU官方驅動；2) 安裝TV Tuner BDA驅動；3) 安裝MPEG-2解碼器（具DXVA）；4) 在MCE設定電視訊號、EPG與錄影路徑（獨立磁碟最佳）；5) 於顯卡控制台啟用影像加速；6) 測試播放/錄影延遲。設定示例：MCE→設定→電視→設定電視訊號；錄影品質：高。注意：磁碟剩餘空間、避免與系統盤同一顆；最佳實踐：啟用寫入快取、使用鎖扣SATA與有風流硬碟位。

Q4: 如何正確接上前置面板與 USB 針腳？

Q5: 如何更換並固定具鎖扣的 SATA 線與理線？

- A詳: 步驟：1) 確認主板與硬碟端有鎖槽；2) 插入SATA線至聽見卡扣聲；3) 以短路徑走線、避免拉力；4) 用紮帶固定於機殼孔位；5) 開機檢查磁碟是否穩定辨識。設定示例：SATA Mode: AHCI（配合驅動）。注意：避免急彎、不要與高溫線束捆綁；最佳實踐：為錄影/系統分盤各用不同通道，降低干擾。

Q6: 如何處理顯卡與機殼風扇的干涉？

- 學習階段: 核心
- 關聯概念: B-Q7, A-Q9, D-Q1

Q7: 如何清理灰塵並加裝濾網避免蚊蟲進入？
- A簡: 關機放電，使用防靜電工具清潔鰭片與風扇，加裝可拆洗濾網，建立定期保養。
- A詳: 步驟：1) 斷電、按電源釋放餘電；2) 用防靜電刷/手動吹球清潔鰭片與電源進氣；3) 固定風扇葉片再清潔；4) 安裝磁吸/卡扣濾網於進氣口；5) 設保養週期（如每月）。設定示例：前進氣裝網、後出風不裝，維持正壓。注意：避免高壓空氣直吹風扇致回授；最佳實踐：室內防蚊與環境整潔同步。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q9, A-Q10, D-Q2

Q8: 如何調整風扇曲線與塗抹導熱膏降低噪音？
- A簡: 以PWM/DC曲線壓低常用轉速，確保溫度門檻；導熱膏薄塗均勻，改善接觸降低溫度。
- A詳: 步驟：1) BIOS→Q-Fan/Smart Fan設曲線（如40℃ 30%、70℃ 70%）；2) 以軟體監控溫度/轉速；3) 重塗導熱膏（豌豆/十字法），對角鎖緊散熱器；4) 壓力測試校正曲線。設定示例：CPU Fan Control Mode: PWM；Fan Slope調至溫和。注意：過低轉速恐致熱積累；最佳實踐：CPU/GPU各自優化，箱扇維持穩定風道。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q11, A-Q12, D-Q10

Q9: 硬體變更後如何完成 Windows XP 啟用？

- A詳: 步驟：1) 開啟「開始→執行→oobe/msoobe /a」檢查狀態；2) 選擇網路或電話啟用；3) 網路失敗則撥打自動語音，輸入安裝ID；4) 取得確認ID並輸入完成啟用。設定示例：確保時間同步、停用攔截。注意：OEM版本與主機板綁定限制；最佳實踐：保留授權資料與購買憑證，重大更換前先備援。
- 難度: 初級
- 學習階段: 基礎

Q10: 對調完成後如何做系統驗證與壓力測試？

### Q&A 類別 D: 問題解決類

Q1: 顯示卡散熱片卡到機殼風扇怎麼辦？
- A簡: 量測後改用薄型/移位風扇，調整理線與硬碟籠；必要時換短卡或相容機殼。
- A詳: 症狀：側板蓋不上、風扇葉片碰撞聲。原因：顯卡長/厚超出機殼淨空、側板風扇過厚。步驟：1) 移除或改薄側板風扇；2) 改前置進氣；3) 調整線材與移除干涉籠位；4) 選短板/單槽顯卡或更換機殼。預防：購前比對尺寸、預留餘量10–20mm、避免側板凸扇設計於窄殼。
- 難度: 中級
- 學習階段: 核心

Q2: 機殼大風扇吸進蚊蟲與灰塵怎麼解？
- A簡: 安裝可拆洗濾網、建立正壓氣流，定期清潔；必要時加紗網或環境防蚊。
- A詳: 症狀：機內發現昆蟲/大量積塵。原因：大面積進氣無濾網、負壓吸入縫隙。步驟：1) 安裝進氣濾網；2) 調整進風量略大於出風量；3) 定期清潔；4) 環境防蚊。預防：選帶濾網機殼、用磁吸濾網便於清洗、避免開放式側板。 
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q10, B-Q8, C-Q7

Q3: 清潔時吸塵器沒電或工具不足怎麼辦？

- A詳: 症狀：無法即時除塵。原因：工具未充電或欠缺。步驟：1) 斷電；2) 防靜電刷清鰭片；3) 吹球/低壓空氣輔助；4) 固定風扇葉片避免回生電壓；5) 事後準備工具（防靜電刷、吹球、濾網、紮帶）。預防：保養排程、工具定期檢查充電、避免使用家用大吸力直接吸主板。

- 學習階段: 基礎
- 關聯概念: B-Q9, C-Q7, A-Q11

Q4: SATA 接頭一碰就掉的排除方法？

- A詳: 症狀：輕碰線材硬碟離線。原因：無鎖扣、線材張力大、插座磨損。步驟：1) 更換鎖扣線；2) 調整短直走線、用紮帶固定；3) 檢查插座緊密度；4) 監看SMART重試錯誤。預防：安裝時保留迴轉餘裕、避免線材受壓、定期重插檢查。

Q5: MCE/XP 提示硬體變更需啟用時如何處理？
- A簡: 先嘗試線上啟用，不行改電話啟用；確保日期正確與網路/防火牆設定無阻。
- A詳: 症狀：登入時要求啟用。原因：主機板/CPU/網卡變更超過門檻。步驟：1) oobe/msoobe /a檢視；2) 線上啟用；3) 失敗則電話輸入安裝ID，取得確認ID；4) 重新啟動。預防：保留授權與購買憑證，重大升級前評估版本限制（OEM vs 零售），必要時先停用授權或備援系統。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, B-Q2, C-Q9

Q6: 前置面板線裝反導致燈不亮、USB 不能用怎麼辦？
- A簡: 依手冊重接LED極性與USB針腳，開機測試；必要時更換標示清楚的轉接線。
- A詳: 症狀：指示燈不亮、USB無反應。原因：LED極性接反、USB插錯排或誤插供電。步驟：1) 手冊比對針腳；2) LED正負對正；3) USB 9-pin依VCC/D-/D+/GND正確接入；4) 測試裝置管理員。預防：線材貼標、拍照留存、使用單一多合一前置I/O模組減少錯誤。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q6, B-Q5, C-Q4

Q7: USB 鍵盤在開機階段按不下去怎麼辦？
- A簡: 改用PS/2鍵盤進BIOS，啟用Legacy USB或更新BIOS；避免不相容KVM/延長線。
- A詳: 症狀：無法在啟動畫面輸入。原因：Legacy USB關閉、KVM不相容、供電不足。步驟：1) 接PS/2；2) 進BIOS啟用Legacy USB；3) 更新BIOS；4) 直連主機板USB。預防：保留PS/2鍵盤、安裝前預設啟用Legacy、選擇相容KVM。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q7, B-Q4, C-Q2

Q8: 對調後 MCE 不能看電視或錄影中斷怎麼辦？
- A簡: 重新安裝TV Tuner驅動、設定頻道與解碼器；檢查磁碟速度與SATA連接穩定。
- A詳: 症狀：黑畫面、找不到裝置、錄影斷檔。原因：BDA驅動遺失/不相容、解碼器未裝、磁碟I/O不穩。步驟：1) 裝正確BDA驅動；2) 設定訊號來源與掃描；3) 安裝支援DXVA解碼器；4) 測磁碟速與更換鎖扣SATA線。預防：建立驅動包、分盤錄影、固定更新策略。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q14, C-Q3, A-Q8

Q9: 對調後電腦無法開機或不斷重啟怎麼診斷？
- A簡: 依POST蜂鳴與最小化系統法排除：檢查電源、RAM、顯卡、儲存模式與短路。
- A詳: 症狀：黑屏、循環重啟。原因：接線錯誤、支撐柱短路、RAM未插牢、儲存模式驅動不符（0x7B）。步驟：1) 最小化：只留CPU/RAM/顯卡；2) 檢查24+8pin電源；3) 重插RAM單條啟動；4) 檢查主機板支撐柱位置；5) 儲存模式改IDE或預裝AHCI驅動。預防：安裝前對位支撐柱、拍照紀錄接線、按步驗證。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q13, B-Q12, C-Q10

Q10: 換了低功耗 CPU 卻仍不安靜的原因與對策？
- A簡: 風扇曲線激進、散熱接觸不良、風道混亂或振動傳導；需調曲線、重上膏與理線減震。
- A詳: 症狀：空閒仍吵。原因：風扇轉速過高、導熱膏塗不良、線材阻風、硬碟/風扇共振。步驟：1) 調整PWM曲線；2) 重新塗膏安裝散熱器；3) 理線優化風道；4) 加貼防震墊或更換低噪風扇；5) 更新BIOS以改善風扇控制。預防：選大面積散熱器、良好風道與濾網、建立定期清潔。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q11, C-Q8, A-Q12
