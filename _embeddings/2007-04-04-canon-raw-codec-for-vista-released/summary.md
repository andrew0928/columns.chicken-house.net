# Canon RAW Codec for Vista 出來了..

## 摘要提示
- Canon RAW Codec: Canon 於 2007/3/30 發佈了 Windows Vista 專用的 RAW 影像解碼器。
- WIC/WPF 整合: 有了官方 Codec 之後，WPF 透過 WIC 便能直接預覽與處理 Canon RAW 檔。
- 支援格式: 首版僅支援 .CR2 副檔名，尚未涵蓋早期機種使用的 .CRW。
- 老機種困境: 使用 G2 等老相機的使用者暫時無法受惠，作者因此小小哀號。
- 未來期望: 作者推測 .CRW 的解碼器應該也會推出，只能再耐心等待。
- 資訊來源: 新聞來自 Microsoft PIX 團隊 3/30 的官方部落格貼文。
- 心情轉折: 從抱怨缺乏 Codec 到發現已經發佈，情緒由不滿轉為驚喜。
- 技術背景: 桌面端 RAW 支援與 .NET 圖形應用程式開發息息相關。

## 全文重點
作者先前才在抱怨 WPF（Windows Presentation Foundation）的 WIC（Windows Imaging Component）無法直接讀取 Canon 相機的 RAW 檔案，沒想到三天前（2007/3/30）Canon 就悄悄推出了專為 Windows Vista 打造的官方 RAW Codec。這項解碼器一安裝，Vista 系統本身以及所有透過 WIC 呼叫影像解碼流程的程式（包含 WPF 應用或資源管理員縮圖檢視）便能立即辨識並顯示 Canon RAW 格式，對攝影愛好者與 .NET 圖形開發者皆是一大福音。

然而初版僅支援新世代相機常見的 .CR2 副檔名，並未涵蓋較早期機種（例如 PowerShot G2）所產生的 .CRW 檔案。作者雖對 .CR2 的到來感到欣喜，仍難掩對老相機被「放生」的失落感，只能樂觀推測 Canon 終究會推出 .CRW 的解碼器，並決定再等等看。整篇文章氛圍介於吐槽與欣慰之間，既呈現了開發者對系統相容性的小牢騷，也流露出對廠商終於動作的感謝。

## 段落重點
### Canon RAW Codec 推出
作者引用 Microsoft PIX 團隊於 2007/3/30 公佈的消息，確認 Canon 已正式釋出 Windows Vista 專用的 RAW Codec。安裝後，Vista 以及所有依賴 WIC 的應用程式（包含 WPF）都能原生顯示與處理 Canon RAW 檔，使得先前必須額外轉檔或倚賴第三方工具的窘境獲得解決。對攝影同好、影像工作者及 .NET 圖形開發者而言，這代表系統層級的支援終於補齊，省去大量時間與維護成本，也讓圖片工作流程更流暢一致。

### 對 .CRW 支援的期望
令人遺憾的是，首版解碼器僅支援較新款數位單眼或高階機種採用的 .CR2 檔案格式，早期 PowerShot G2 等相機使用的 .CRW 仍被排除在外。作者對此哭喊「啊啊啊啊啊」，在歡欣之餘表達對老機身的同情與不平；但也從積極角度思考，既然 .CR2 已經開了頭，.CRW 的 Codec 理論上只是時間問題。文章最後留下「再等一等好了」的語句，折射出開發者對廠商後續支援的期望與無奈。
