# Ouch, 為什麼沒有 c720w ...

## 摘要提示
- Dopod升級計畫: 多普達宣布 U1000、P800W、CHT9000、CHT9100 四款手機可免費升級 Windows Mobile 6。  
- C720W缺席: 作者所用的 C720W 不在官方名單，引發強烈失落與不滿。  
- 同型機對比: 國外 T-Mobile 版同款機已提供 WM6 升級，凸顯政策差異。  
- 非官方ROM: 作者已取得 T-Mobile 版升級檔，卻因風險高而遲遲未刷。  
- 升級風險: 私自升級恐失保固、無法還原、甚至刷機失敗變磚。  
- 社群求援: 作者徵求「勇者」分享成功升級經驗，降低疑慮。  
- 對HTC喊話: 期待 HTC／Dopod 盡快釋出 C720W 官方 WM6 ROM。  
- 用戶心聲: 文章反映品牌售後支援落差帶來的失望與訴求。  

## 全文重點
作者注意到多普達（Dopod）官方網站公布的 Windows Mobile 6 免費升級計畫，列出 U1000、P800W、CHT9000 與 CHT9100 等四款機型，但唯獨缺少自己手上的 C720W。此結果令作者十分挫折，更因發現硬體規格相同的 T-Mobile 版已在國外開放 WM6 韌體下載，對比之下顯示問題並非技術層面，而是原廠政策選擇。  
雖然作者已透過朋友取得 T-Mobile 版升級程式，但考量私刷後可能導致內建軟體無法復原、保固立即失效，且缺少官方還原工具，一旦失敗就「欲哭無淚」而不敢嘗試，只好向社群詢問是否有人具體完成升級並分享經驗。文末作者直接對 HTC／Dopod 發聲，希望兩者既已合併為一家，就應正視 C720W 使用者的需求，儘快釋出官方 WM6 ROM，讓所有機型都能受到同等支援。整篇文章展現出使用者面對產品支援不一致時的無奈與期待，也反映品牌在售後服務上的重要性。  

## 段落重點
### 官方升級計畫與 C720W 缺席
作者引用多普達新聞稿指出，U1000、P800W、CHT9000、CHT9100 可免費升級至 Windows Mobile 6，證明廠商確實著手維護既有客戶；然而名單獨缺 C720W。對於「我的手機被遺忘」的落差感，作者以「很好很好，沒有我這台 C720W 就算了」表達失望，並突顯官方升級策略的差別待遇。

### T-Mobile 提供升級、私刷顧慮與用戶呼籲
作者進一步指出，國外 T-Mobile 版同款機種已正式釋出 WM6 升級檔，足見硬體相容性並非問題。雖手中已有 T-Mobile ROM，卻擔憂刷機後失去保固、內建軟體回不去、缺乏還原方案，因此遲遲不敢行動，只能向社群尋求「勇者」經驗。文章最後直接呼籲 HTC／Dopod 盡速提供 C720W 官方 WM6 ROM，讓用戶能在安全、合法、保固無虞的前提下升級，也反映出消費者對完善售後支援的殷切期待。
