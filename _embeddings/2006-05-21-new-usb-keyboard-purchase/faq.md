# 新鍵盤...

## 問題與答案 (FAQ)

### Q&A 類別 A: 概念理解類

Q1: 什麼是標準鍵盤布局？
- A簡: 常見全尺寸104鍵（ANSI），數字鍵區與方向鍵獨立、功能鍵置頂，按鍵位置穩定，有利肌肉記憶並降低誤觸。
- A詳: 標準鍵盤布局多指ANSI 104鍵的全尺寸配列：主鍵區、方向鍵區與數字鍵區分離，F1–F12位於上方，Insert/Del/Home/End/PgUp/PgDn等為三列或兩列集成。此布局長期被廣泛使用，對盲打者特別友善，因為各區位置固定、鍵距一致，能有效降低視線移動與誤觸率。對需要頻繁輸入數字或進行導航操作的使用者，分明的分區能兼顧效率與準確度。若布局被挪動、縮減或合併，常會破壞肌肉記憶，導致誤按與學習成本增加。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, B-Q11, C-Q2

Q2: 為什麼選擇USB鍵盤而非PS/2？
- A簡: USB支援即插即用，方便在多台電腦間拔插；PS/2常需重開機才偵測，靈活性與便利性不及USB。
- A詳: USB鍵盤遵循HID通用協定，插入後能被主流作業系統即時偵測與載入通用驅動，免重開機即可使用。這對經常在多台主機或筆電間切換的使用情境尤其友善。相較之下，PS/2鍵盤多需在開機自檢階段初始化，若於系統運行中才插上，常無法被識別，甚至可能造成主機板風險。因此在便利性、相容性與日常操作效率上，USB通常優於PS/2，也更符合「拔來拔去」的需求。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q1, B-Q2, C-Q5, D-Q1

Q3: 什麼是筆電式鍵盤結構？
- A簡: 多為剪刀腳薄膜結構，鍵程短、回饋穩定、噪音低，手感接近筆電，適合習慣筆電打字者。
- A詳: 筆電式鍵盤常見為剪刀腳（Scissor switch）配薄膜開關：十字連桿提供側向導向與支撐，使按鍵上下穩定且鍵程短，回彈清楚、噪音較低。此結構有利薄型化與長時間辦公，手感接近多數筆電。相對於傳統薄膜鍵盤，它在鍵帽晃動控制與一致性上更佳；相對機械式鍵盤，手感更輕巧、聲音更安靜，但客製化與可維修性較低。若你日常以筆電為主，轉用此類桌上鍵盤能減少適應成本。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q5, C-Q7, A-Q13

Q4: 何謂鍵盤外觀簡約設計？
- A簡: 移除多餘多媒體鍵與燈效，保留必要按鍵與清爽線條，降低誤觸與視覺干擾，重視耐看與實用。
- A詳: 簡約設計強調實用與專注。它通常維持標準配列，避免增設大型多媒體或巨集鍵，以減少誤觸與學習成本；亦避開過度的燈效與裝飾，讓視覺更乾淨、長時間使用更舒適。這樣的外觀也較容易融入辦公或居家環境，不搶戲、不分散注意力。對重視打字效率、肌肉記憶與長時間工作者，簡約外觀往往意味著更直覺的操作與更穩定的體驗。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q10, A-Q12, C-Q1

Q5: 標準布局與改動布局的差異是什麼？
- A簡: 標準布局分區清楚、鍵距穩定；改動布局常縮排或挪位，破壞肌肉記憶，提升誤觸與學習成本。
- A詳: 標準布局（如ANSI 104）將主鍵區、方向鍵區、數字鍵區清楚分隔。改動布局則可能壓縮或合併這些區域，例如把方向鍵嵌入主鍵區、縮短Shift/Enter、或把多媒體與電源鍵放在右上群組附近。這會影響盲打定位，特別在視線未完全回到鍵盤時，容易按錯（如Pause附近誤按到電源鍵）。改動布局適合攜帶或空間受限，但代價是更高的適應曲線與錯誤風險。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q1, B-Q11, D-Q5

Q6: NumLock指示燈過亮有何影響？
- A簡: 過亮白光LED在暗環境易炫光，造成刺眼與視覺疲勞，影響長時間使用的舒適與專注。
- A詳: 高亮度、集中光點的白光LED在低環境光下會產生顯著炫光，直視時刺眼，側視也可能造成干擾。長時間暴露會帶來視覺疲勞、分心，甚至讓人下意識調整姿勢以避光，進一步影響工學。若鍵盤LED亮度無法透過軟體調整，可用霧面膠帶或半透明材質擴散，或改變鍵盤擺位與視角，避免直視光源。購買時也可留意指示燈亮度與位置設計。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q10, C-Q4, D-Q4

Q7: 為何常以ThinkPad鍵盤當作手感參考？
- A簡: 其穩健剪刀腳、鍵帽穩定與一致回饋受好評，常被視為筆電鍵盤標杆，用於挑選時對照。
- A詳: 多數ThinkPad機型以穩定的剪刀腳結構、恰當鍵程與一致性的敲擊回饋著稱，長期被辦公族群與文字工作者採用。當選購桌上鍵盤時，使用者常以熟悉的ThinkPad手感作為標杆，衡量鍵程、回彈、鍵帽穩定與整體節奏是否接近，藉此降低適應成本。雖然不同世代與型號手感仍有差異，但它作為「參考物」有助於更快速決策。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q3, A-Q13, C-Q10

Q8: USB鍵盤的核心價值是什麼？
- A簡: 跨設備即插即用、驅動通用、供電穩定、擴充彈性高，日常切換與測試便利。
- A詳: USB屬通用序列匯流排，鍵盤採HID類別，主流系統內建驅動即可用，不需額外安裝。它支援熱插拔，方便在桌電、筆電、切換器、擴充埠之間靈活使用。供電由USB埠提供，對大多數有線鍵盤足夠穩定。加上配件與線材普及，若需延長線、加裝集線器或切換器也相對容易。這些特性構成USB鍵盤在便利性與相容性上的核心價值。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q1, C-Q5, D-Q2

Q9: 為什麼選購前要明確需求條件？
- A簡: 先列介面、布局、外觀等條件，可快速篩選、避免被噱頭干擾，提升滿意度與使用效率。
- A詳: 市面鍵盤眾多，從燈效、多媒體鍵到造型差異繁多。若事先明確界定「USB介面」「標準布局」「簡約外觀」等必要條件，可在眾多選項中快速排除不合適款式，把注意力放在真正重要的手感、鍵距、品質與價格上。這也能降低衝動購買與日後後悔的機會，讓選擇更貼近實際使用情境與長期需求。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q1, A-Q4, A-Q5

Q10: 3C賣場鍵盤常見設計問題是什麼？
- A簡: 常強調多媒體鍵與燈效，伴隨布局壓縮或挪位，與實用需求不符，長期使用易誤觸。
- A詳: 為吸引目光，許多鍵盤增加多媒體鍵、巨集鍵與強烈燈效，並為了縮小尺寸或強化主題設計而改動配列（壓縮Enter、合併方向鍵等）。雖能滿足特定族群，但對以效率與穩定為先的使用者，這些改動經常帶來誤觸與學習成本。對長時間打字或辦公者，簡單、標準、耐用往往更合適。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q4, A-Q5, D-Q5

Q11: 何謂全尺寸、無數字鍵(TKL)、緊湊型鍵盤？
- A簡: 全尺寸含數字鍵區；TKL去除數字區；緊湊型進一步壓縮或合併方向鍵與功能區。
- A詳: 全尺寸（約104鍵）保留所有分區，適合大量數字輸入與辦公。TKL（約87鍵）刪除數字鍵區，節省桌面空間並拉近滑鼠距離，舒適性較佳。緊湊型（如75%、65%、60%）進一步壓縮或合併方向鍵與功能列，提升攜帶性，但需透過組合鍵或層切換，並可能增加誤觸與學習成本。選擇時需衡量空間、攜帶、輸入需求與錯誤容忍度。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q1, A-Q5, C-Q2

Q12: 為什麼容易誤按Pause與關機鍵？
- A簡: 改動布局常把電源/睡眠鍵置於右區，靠近Pause/Del等；視線轉移時易誤觸，造成關機或睡眠。
- A詳: 非標準配列有時會在方向鍵、主鍵區右上角加入電源、睡眠、喚醒鍵，位置與Pause、Del、PgUp/PgDn接近。視線不在鍵盤、憑肌肉記憶操作時，極易按錯，嚴重者可能直接觸發關機或睡眠，造成資料中斷。對策包括選擇標準配列、在作業系統停用或重映射電源鍵、或調整鍵盤擺位與使用習慣，以降低風險。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q3, D-Q3, A-Q5

Q13: 如何客觀評估鍵盤手感？
- A簡: 從鍵程、觸發力、穩定性、回彈與噪音評估，並以長時間試打與錯誤率紀錄佐證。
- A詳: 手感評估可拆成指標：鍵程（短長與一致性）、觸發力（是否疲勞）、鍵帽穩定（晃動與偏心）、回彈節奏（是否黏滯）、噪音（環境友善度）。建議進行10–15分鐘連續盲打，觀察速度與錯誤率、手腕與肩膀緊繃度，並在不同姿勢與角度測試。若有參考鍵盤（如熟悉的筆電鍵盤），可對照確認適應成本。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q10, B-Q5, A-Q7

Q14: 價格在選購上的意義是什麼？
- A簡: 價格反映用料做工與品牌，但不等同手感；符合需求且價位合理（如500元內）即具高性價比。
- A詳: 價格通常影響結構用料、製造精度、穩定性與保固，但未必直接對應個人手感。關鍵在於是否符合自身需求（介面、布局、手感、外觀）與使用情境（辦公、遊戲、攜帶）。若在預算內達到足夠品質與良好手感，即屬高性價比。比價時也應留意售後、鍵帽材質與長期耐用性，而非只看華麗外觀或宣傳規格。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q1, C-Q10, A-Q9

Q15: 為什麼有些人偏好白光LED而另一些人排斥？
- A簡: 白光LED清晰現代，但亮度高易刺眼；偏好取決於環境光、敏感度與審美需求。
- A詳: 白光LED指示清楚、風格現代，能在明亮環境保持可視；但其點光源在昏暗環境易產生炫光，對光敏感者特別刺激。是否喜歡取決於使用環境（辦公照明/夜間）、個人對亮度的耐受、與對外觀的偏好。若經常夜間使用或面向LED，選擇柔和或較暗指示燈、或預留遮光對策，能兼顧美觀與舒適。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q6, B-Q10, C-Q4

### Q&A 類別 B: 技術原理類

Q1: USB鍵盤如何運作？
- A簡: 透過HID協定，主機枚舉裝置、讀取描述符，依報告格式接收鍵值，使用通用驅動即時處理。
- A詳: USB鍵盤屬HID（Human Interface Device）類別。插入後，主機偵測到電氣連線並進行枚舉，讀取裝置與報告描述符，瞭解鍵值報告格式。鍵盤以中斷傳輸定期回報按鍵狀態，作業系統以內建HID驅動解析為虛擬鍵事件，再交由應用程式處理。由於協定通用，通常免安裝專屬驅動，即可即時使用。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q8, B-Q8, D-Q2

Q2: PS/2與USB在熱插拔上的差異原理？
- A簡: PS/2直連控制器且無完善熱插拔保護；USB具枚舉流程與電氣保護，設計為即插即用。
- A詳: PS/2為較早期同步串列介面，鍵盤在開機自檢時初始化並由主機板控制器直接管理，運作中熱插拔可能因缺電氣保護與協定流程而無法辨識，甚至造成硬體風險。USB則內建供電偵測、連線狀態監測與枚舉協定，搭配HID類別通用驅動，支援即插即用且更安全。因此日常拔插與跨設備使用，USB更具優勢。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q2, D-Q1, B-Q8

Q3: 鍵盤矩陣掃描機制是什麼？
- A簡: 以行列矩陣輪詢按鍵閉合，控制器判斷並輸出掃描碼，兼顧線材節省與功能完整。
- A詳: 鍵盤將按鍵排列成行列矩陣，控制器依序驅動行線並讀取列線，偵測哪一組交點被導通。此設計可大幅減少導線數與成本。為避免同時多鍵產生鬼鍵（ghosting），高階設計會加入二極體改善阻隔。掃描結果被轉換為掃描碼，再交由主機映射為虛擬鍵。掃描頻率、除抖與抗干擾等，都影響最終輸入品質。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: D-Q8, B-Q4, B-Q15

Q4: 按鍵防彈跳(debouncing)如何實現？
- A簡: 透過硬體RC濾波或韌體延時，忽略接點抖動，確保一次按鍵只被記錄一次。
- A詳: 機械接點在閉合/斷開瞬間會快速抖動，產生多次高低變化。防彈跳可在硬體以RC濾波與施密特觸發等平滑訊號，或在韌體以時間窗延遲與狀態機判斷，當穩定超過門檻才認為有效。良好的除抖能降低連點誤判，提升輸入可靠性，對快速打字與遊戲尤為重要。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: D-Q8, B-Q3

Q5: 剪刀腳機構的技術特性？
- A簡: 十字連桿導向薄膜開關，鍵程短、側向穩定、手感一致，兼顧薄型與靜音需求。
- A詳: 剪刀腳在鍵帽與薄膜開關間加入交叉連桿，提供導向與支撐，減少鍵帽左右晃動，讓短鍵程也能保持穩定。其結構讓鍵盤可做得更薄、更安靜，適合長時間辦公。限制是可維修性較低、客製化與可替換零件較少；但對習慣筆電手感者，過渡到桌上型時可快速上手。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q3, A-Q13, C-Q7

Q6: NumLock等指示燈如何被控制？
- A簡: 主機透過協定設定LED狀態，鍵盤控制器驅動LED；多數裝置亮度固定，難以軟體調光。
- A詳: 在USB HID中，主機可送出Set_Report變更LED位元（NumLock、CapsLock、ScrollLock），鍵盤控制器據此驅動LED腳位與限流電路。一般來說只有開/關，不提供亮度調整；亮度取決於LED特性與驅動電流。因此若覺得過亮，通常需以遮光、擴散或調整視角來改善，而非軟體設定。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q6, C-Q4, D-Q4

Q7: 鍵盤上的電源/睡眠鍵與ACPI如何互動？
- A簡: 特殊鍵送出掃描碼，作業系統轉為ACPI事件，依電源設定執行關機或睡眠等動作。

Q8: USB即插即用流程為何？

Q9: 筆電外接鍵盤時輸入如何合併？

Q10: LED眩光的人因工程要點？
- A簡: 高亮點光源在暗環境易炫光；視角、擴散與距離影響舒適度，需減少直視與反射。
- A詳: 炫光源於高亮、面積小的點光在視網膜形成強刺激，暗環境對比越大越明顯。改善策略包括：擴散（霧面膠帶、擴散片）、調整視角與擺位避免直視、拉開距離降低入眼照度、或提高環境光讓對比下降。設計上可選用亮度適中的LED、擺放在不直視的側向位置，減少長時間使用的不適。

Q11: ANSI、ISO、JIS鍵盤布局差異？
- A簡: ANSI短Enter長左Shift；ISO大Enter、短左Shift且多一鍵；JIS含日文專鍵與不同空白鍵。
- A詳: ANSI（美式）特色是長左Shift、短Enter一排；ISO（歐洲）改用倒L形大Enter，左Shift縮短並在旁增一鍵；JIS（日式）加入假名/切換鍵、空白鍵較短，周邊多功能鍵。差異影響鍵帽尺寸、指法與肌肉記憶。跨布局轉換需適應，選購前應確認自身習慣與實際鍵位。

Q12: 數字鍵區與方向鍵分區設計理由？

Q13: 鍵盤指示燈硬體電路如何設計？
- A簡: 以限流電阻與驅動晶體管控制LED通斷；亮度由電流決定，通常不支援調光。
- A詳: 指示燈電路包含LED本體、限流電阻與由微控制器IO或晶體管開關驅動。亮度與色溫由LED規格與驅動電流決定。多數鍵盤只提供開關，不具PWM調光。若要改變亮度，需更動硬體（調整電阻或更換LED），涉及焊接與保固風險，不建議一般使用者嘗試。實務上多以遮光與擺位改善。

Q14: 透過USB集線器/切換器切換鍵盤的原理？

- A詳: 簡單切換器實體切換上游連線，每次切換皆重新枚舉，可能短暫無輸入；進階KVM採協定模擬，對主機維持「常連接」假象，切換更順但成本高。不同實作會影響NumLock狀態同步與熱鍵行為。選購時應留意支援的HID佔線、延遲與狀態同步能力。

Q15: 鍵值掃描碼如何映射為OS行為？

### Q&A 類別 D: 問題解決類

Q1: 插上PS/2鍵盤無反應怎麼辦？
- A簡: PS/2多不支援熱插拔；關機再接、進BIOS檢查、換鍵盤或改用USB以改善。
- A詳: 症狀：系統已開機後插入PS/2鍵盤無反應。原因：PS/2需於開機自檢初始化，運行中插入多半不被識別，且缺電氣保護。解法：1) 關機斷電再接上。2) 進BIOS確認PS/2啟用。3) 交叉測試另一把鍵盤或主機。預防：避免運行中插拔PS/2；若需常切換，改用USB鍵盤或切換器。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q2, B-Q2, C-Q5

Q2: USB鍵盤偶爾失靈或延遲怎麼排查？
- A簡: 換USB埠、直連主機、關閉省電、更新驅動並重插重新枚舉，排除Hub與供電問題。
- A詳: 症狀：輸入延遲、偶發失靈。原因：Hub供電不足、線材接觸不良、USB省電、系統負載高。解法：1) 直連主機背板USB。2) 更換線材或避免延長線。3) 關閉USB選擇性暫停與裝置省電。4) 重新插拔觸發枚舉。5) 更新晶片組/USB驅動。預防：避免串聯過多Hub，使用品質良好線材與切換器。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q8, C-Q9, C-Q5

Q3: 誤按關機鍵導致關機如何避免？
- A簡: 在電源選項將其設為不動作，或以重映射工具停用；同時調整擺位減少誤觸。
- A詳: 症狀：在右上群組鍵附近誤觸電源/睡眠鍵。原因：改動配列將電源鍵置於易誤觸區。解法：1) 電源選項中把電源鍵行為改為「不執行任何動作」。2) 以重映射工具停用或改為無害按鍵。3) 調整鍵盤角度與位置。預防：選購標準配列、避免將易誤觸鍵與常用群組相鄰，或於鍵帽上加凸點標記區別。

- 學習階段: 基礎
- 關聯概念: A-Q12, C-Q3, B-Q7

Q4: NumLock燈太亮刺眼如何處理？
- A簡: 以霧面貼擴散、調整鍵盤視角與距離、增加環境光；必要時改選低亮度款。
- A詳: 症狀：直視指示燈感到刺眼、分心。原因：點光源亮度高、對比強。解法：1) 貼霧面膠帶或半透明紙擴散。2) 調整鍵盤位置避免直視。3) 增加環境光降低對比。4) 重新選購較柔和指示燈的型號。預防：購買前在實際環境檢視指示燈亮度與角度。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q6, B-Q10, C-Q4

Q5: 經常按錯鍵的原因與解法？
- A簡: 多為非標準配列、鍵距差異與視線不在鍵盤；改用標準配列、練盲打與重映射能改善。
- A詳: 症狀：方向鍵、群組鍵、Pause/Del等常誤按。原因：配列改動、鍵帽尺寸差異、指示燈干擾。解法：1) 換標準配列。2) 盲打訓練與降低視線依賴。3) 以重映射調整問題鍵位。4) 在特定鍵帽加凸點。預防：選購前做實機試打與配列檢查，避免將特殊鍵置於常用區旁邊。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, C-Q6, B-Q15

Q6: 外接鍵盤與筆電鍵盤衝突或重複輸入？
- A簡: 系統會合併輸入；檢查語系與黏滯鍵設定，必要時暫時停用故障裝置以排除。
- A詳: 症狀：重複輸入或行為異常。原因：語系設定不一致、輔助功能（黏滯鍵）誤開、內建鍵盤故障連發。解法：1) 統一輸入語系。2) 關閉黏滯鍵/範圍鍵。3) 在裝置管理員暫時停用可疑裝置測試（注意外接滑鼠/鍵盤可用性）。預防：保持設定一致，避免潮濕與髒污導致鍵盤自發輸入。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q9, C-Q9, C-Q5

Q7: 鍵帽晃動或鍵程不均怎麼辦？
- A簡: 可能為剪刀腳鬆動或積塵；進行清潔與檢查，必要時更換鍵帽或整把鍵盤。
- A詳: 症狀：某些鍵左右晃動大、手感不一致。原因：剪刀腳扣具鬆脫、鍵帽磨損、灰塵堆積。解法：1) 斷電清潔、氣吹鍵縫。2) 檢查鍵帽是否卡住。3) 參考型號教學，小心重新扣回。4) 嚴重磨損時更換鍵帽或整把鍵盤。預防：使用防塵罩、避免油污堆積，定期清潔維護。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q7, B-Q5, D-Q8

Q8: 發生按鍵連點/不觸發如何處理？
- A簡: 可能為接點彈跳或髒污；清潔、調整重複延遲，仍故障則維修或汰換。
- A詳: 症狀：輕按即多字、或需重按才反應。原因：接點抖動、灰塵、結構疲勞。解法：1) 清潔鍵帽與周邊。2) 系統調整「字元重複延遲/速率」。3) 交叉測試另一台電腦。4) 持續異常可能為硬體故障，送修或汰換。預防：維持鍵盤清潔、避免液體滲入、減少過度用力敲擊。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q4, B-Q3, C-Q7

Q9: 使用KVM或切換器後NumLock狀態錯亂？
- A簡: 切換時狀態同步被打斷；設定KVM同步或手動矯正，必要時更換支援度較佳機型。
- A詳: 症狀：切換主機後NumLock開關相反或不一致。原因：簡易切換器在切換時重新枚舉，未同步LED狀態；或KVM熱鍵衝突。解法：1) 啟用KVM的狀態同步功能。2) 調整或關閉切換熱鍵。3) 切換後手動矯正狀態。4) 選購具協定模擬的KVM。預防：在購買前確認KVM對HID狀態的處理方式。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q14, C-Q8, C-Q5

Q10: 指示燈不亮或不受控如何診斷？
- A簡: 檢查OS LED設定、換埠重裝與交叉測試；若他機也失效，可能為硬體故障需維修。
- A詳: 症狀：Num/Caps/Scroll燈不亮或無法切換。原因：系統設定、驅動異常、KVM干擾或硬體損壞。解法：1) 測試切換對應鍵看OS狀態是否改變。2) 換USB埠、重新插拔。3) 另一台電腦交叉測試。4) 移除中間Hub/KVM。5) 仍不亮多為硬體問題。預防：避免拉扯線材與濫用切換器，選擇做工可靠的鍵盤。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q6, C-Q9, D-Q2
