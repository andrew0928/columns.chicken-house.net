# 又變成水電工了..

## 問題與答案 (FAQ)

### Q&A 類別 A: 概念理解類

A-Q1: 什麼是家庭結構化佈線？
- A簡: 家庭結構化佈線以星狀拓撲集中管理網路、電話、電視等弱電，配合機櫃、配線架與標準線材，提升穩定、擴充與維護效率。
- A詳: 家庭結構化佈線是將網路、電話、CATV 等弱電系統，依標準化做法以「星狀拓撲」集中到弱電機櫃或配線間，由配線架、交換器、分配器統一管理。它強調模組化端接（RJ45、F頭、RJ11）、一致的線序、標籤與文檔化，並預留擴充。好處包含：更容易擴充（新增孔位、改用途）、更好維護（集中測試、容易更換）、更穩定（線長可控、干擾可管控）、美觀安全（線材藏天花板與管道）。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q2, A-Q3, A-Q16, B-Q1

A-Q2: 為何應在裝潢隔間時一次把弱電線路拉好？
- A簡: 裝潢期打開牆體與天花板，成本低、施工便利，能一次完成星狀佈線與預留，避免日後高成本重做。
- A詳: 裝潢隔間期間是弱電佈線最佳時機，因為牆體開放、管線可重新配置。此時可規劃星狀拓撲、加大管徑、預留備用線與空管，將網路、電話、CATV 統一拉回機櫃。一次完成可避免日後動工造成拆牆、粉塵與高昂費用，也能兼顧位置、電源與通風等需求。長期來看，總成本更低，擴充也更容易。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q1, A-Q3, C-Q2

A-Q3: 什麼是星狀拓撲？為何適合家庭佈線？
- A簡: 星狀拓撲是每條線獨立回到中心機櫃，方便故障隔離、擴充彈性高，最適合家庭弱電集中管理。
- A詳: 星狀拓撲指每個房間或插座都有獨立線路回到中心（機櫃或配線間），而非串接。優點是每條線可獨立測試、替換或改用途（如電話改網路），利於加裝總機、強波器或交換器集中管理。雖然電話與CATV串接也能用，但星狀有更高靈活性與維護性，特別是未來需求變動頻繁的家庭環境。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q4, B-Q2, C-Q2

A-Q4: 為何電話與CATV通常不需星狀，仍建議拉星狀？
- A簡: 星狀讓電話可接總機、CATV可接強波/分配器集中管理，日後調整分機、訊號更方便，維護也更簡單。
- A詳: 傳統電話與CATV多採串接，但星狀讓每一點都能回到中心，電話可接入PBX總機進行分機分配、轉接與門口機整合；CATV則可集中接分配器與強波器，針對每路調整衰減與等化。當出現故障或要改用途（如電話改RJ45網路），只需在機櫃端改接，無須開牆施工，總體可維護性大幅提升。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q5, B-Q6, C-Q6

A-Q5: 什麼是CAT5e？適合家庭使用嗎？
- A簡: CAT5e是乙太網路銅纜標準，支援1Gbps/100m，成本低、相容性好，家庭用網與電話佈線皆適合。
- A詳: CAT5e（Category 5e）是廣泛使用的非屏蔽雙絞線標準，設計支援1GbE在100公尺內穩定傳輸。其性價比高、材料與配件齊全，對於家庭網路、IP設備、NAS、印表機與電話（以RJ11或RJ45端接）都很合適。相較CAT6/6A，它成本更低且施工容錯更大，是多數家庭裝潢的務實選擇。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q18, B-Q16, C-Q3

A-Q6: 為何用CAT5e取代傳統電話線？
- A簡: CAT5e頻寬高、抗干擾佳，端接彈性大；同線可接RJ11當電話，日後改RJ45即成網路孔。
- A詳: 以CAT5e替代電話線能一次鋪管到位，短期用RJ11接電話，長期可改為RJ45當網路孔，保留升級彈性。CAT5e的雙絞結構能降低雜訊與串音，語音品質更穩定；當家庭IP化（VoIP、智慧家電）需求上升，更可直接轉為資料通道，避免重拉線材，降低後續改裝成本。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q3, B-Q3, A-Q7

A-Q7: RJ45與RJ11有何差異？
- A簡: RJ45為8芯網路插頭，RJ11為2-4芯電話插頭；外形、針數與用途不同，不可直接互插混用。
- A詳: RJ45採8P8C，常用於以太網路；RJ11多為6P2C或6P4C，常用於類比電話。RJ45體積較大，針腳更多、對雙絞線挨線規範更嚴謹；RJ11常只用中間2芯。雖可在RJ45插座上布線支持電話用途（用中間對），但物理插頭不同，不建議混用以免損傷彈片或接點。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q3, C-Q3, A-Q12

A-Q8: 每房兩個網路孔加一個電話孔的考量是什麼？
- A簡: 裝置需求多元（電腦、NAS、印表機、影音、遊戲機），兩網一話兼顧現況與擴充，未來可改三網。
- A詳: 隨設備普及（PC、網印、NAS、機上盒、遊戲機、音響串流），單孔常不足。兩個網路孔滿足基本資料設備需求，另留電話孔兼容傳統語音或未來VoIP。若以CAT5e佈線，電話孔日後可換RJ45即變第三網路孔，配置更靈活，減少外露交換器或延長線的雜亂與故障點。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q1, C-Q3, A-Q17

A-Q9: 家用網路基礎設備有哪些？
- A簡: 機櫃、配線架、交換器、路由器、面板插座、線纜與工具；CATV分配器、電話總機依需求選配。
- A詳: 典型設備包含：弱電機櫃（集中與保護）、配線架與理線器（端接管理）、交換器與路由器（資料交換與上網）、面板插座與面板、標籤套件、壓線工具與測試器。若有電視與電話需求，另需CATV分配器/強波器、電話總機（PBX）。監控需求可配DVR/NVR與硬碟，並預留PoE或電源。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q4, C-Q5, B-Q10

A-Q10: 什麼是15U機櫃？家庭適用嗎？
- A簡: 15U機櫃為中小型19吋設備櫃，容納配線架、交換器、DVR/NAS等，集中整線並提升安全與美觀。
- A詳: 「U」為機櫃高度單位（1U=44.45mm）。15U機櫃可安裝19吋標準設備與配件，如配線架、交換器、PDU、DVR/NVR、NAS與小型伺服器。家庭使用可將雜亂線材與分配器集中，減少灰塵與碰觸風險，並整合通風、上鎖與理線。不過需考量散熱、噪音與電力冗餘配置。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q10, C-Q4, A-Q16

A-Q11: 交換器（Switch）與集線器（Hub）差異？
- A簡: Hub廣播轉送、碰撞多；Switch依MAC學習轉發，效率高、可全雙工，是現代家庭的標準選擇。
- A詳: Hub屬共享媒體，所有封包廣播至各埠，易產生碰撞與帶寬共享瓶頸。Switch會學習MAC表，僅將封包送往目的埠，支援全雙工、合併更高背板頻寬，並常具備VLAN、QoS等功能。現今多採交換器，不建議再用Hub，以確保多裝置並發時的效能與穩定性。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q8, D-Q2, C-Q7

A-Q12: 什麼是資訊面板與模組化插座（Keystone）？
- A簡: 面板是牆面裝飾與固定，Keystone模組提供RJ45/RJ11等端口，方便更換、維修與配置彈性。
- A詳: 資訊面板安裝於牆面，承載一至多個Keystone模組。Keystone模組可為RJ45、RJ11、同軸F頭等，採卡扣設計，便於快速更換或改組合。施工以打線工具將雙絞線端接於Keystone，再卡入面板；優點是標準化、易維護、外觀整齊，且可與配線架對應標號一一映射。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q5, B-Q17, A-Q7

A-Q13: 什麼是F頭與CATV同軸線？
- A簡: F頭為CATV常用同軸接頭，搭配75Ω同軸線傳輸電視與數據訊號，需良好壓接與分配。
- A詳: CATV使用75Ω同軸（如RG-6），以F型接頭連接分配器、電視盒、纜線數據機。良好的壓接與端子品質影響回傳與下行訊號強度與雜訊表現。星狀回機櫃可集中接強波器與分配器，精準控制每一路的衰減與隔離，減少雪花、方塊化與頻道干擾。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q5, C-Q6, D-Q3

A-Q14: 什麼是電話總機（PBX）？
- A簡: PBX集中管理多分機與外線，提供轉接、內線撥打、語音功能，家用亦可簡化電話配線。
- A詳: PBX（Private Branch Exchange）連接多條外線與多個內部分機，提供內線互撥、外線轉接、分機代接與語音選單等功能。家庭採星狀佈線可將每房電話回接PBX，靈活分配分機與對應鈴響，未來可整合門口機、語音信箱或VoIP中繼，提升使用彈性與擴充性。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q6, C-Q6, D-Q4

A-Q15: 什麼是DVR/NVR家用監控錄影？
- A簡: DVR錄類比訊號，NVR錄IP影像；皆存入硬碟並可經網路遠端存取，適合集中於機櫃管理。
- A詳: DVR（Digital Video Recorder）接收類比攝影機訊號；NVR（Network Video Recorder）接收IP攝影機串流。兩者皆將影像存入硬碟，支援事件偵測、遠端觀看。集中於機櫃便於電源、網路與散熱統一管理，並可結合UPS與網路隔離，提升可靠度與安全性。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q7, C-Q8, D-Q8

A-Q16: 為什麼所有線路要集中到機櫃？
- A簡: 集中化便於維護、擴充與故障隔離，並能整合散熱、供電與安全，讓居家空間更整潔。
- A詳: 將網路、電話、CATV與監控線路集中到機櫃，能以配線架、交換器、分配器統一配置。出現故障可快速定位到單一線路或端口；新增設備、改用戶型時只需在機櫃端操作。機櫃同時提供通風、上鎖與理線配件，將雜亂設備藏於雜物間，兼顧美觀與安全。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q4, C-Q6, B-Q10

A-Q17: 為什麼要多留網路孔與預留線？
- A簡: 裝置與應用持續增加，預留孔位與線材可降低未來施工成本，避免外露延長線造成不穩定。
- A詳: 隨著NAS、智慧家電、串流播放器、遊戲機與工作站等需求成長，單孔配置容易不足。多留一至兩孔與備用線能在需求改變時快速對應，不必再開牆或拉明線。預留空管亦有助於未來升級（如PoE、CAT6A），兼顧穩定度與擴充性，整體成本更低。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q1, C-Q2, A-Q8

A-Q18: CAT5e、CAT6、CAT6A差異是什麼？
- A簡: CAT5e支援1G/100m；CAT6可10G/短距；CAT6A可10G/100m，屏蔽與外徑更大，施工要求更高。
- A詳: CAT5e適用1GbE至100m，成本低；CAT6提升對高頻干擾控制，可在55m內跑10GbE；CAT6A設計支援10GbE至100m，但線徑大、安裝與轉彎半徑要求更嚴苛。家庭若以1GbE為主，CAT5e性價比高；若長期規劃10GbE與PoE++，可考慮關鍵幹線採CAT6A。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q16, C-Q2, D-Q2

A-Q19: 「像網咖」的家庭網路核心價值是什麼？
- A簡: 高可用、高擴充與集中管理，能同時支援多裝置高流量，維護與升級成本更可控。
- A詳: 類網咖等級的家庭網路並非炫技，而是為了支援多節點同時高速連線（串流、備份、遊戲、工作），並藉由結構化與集中化組態降低維護複雜度。這種設計強調冗餘、標準化與可觀測性，讓未來增設設備、區網分段與安全加固更容易。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q8, C-Q7, D-Q2

A-Q20: 為何需要壓線工具與測試器？
- A簡: 專業工具確保端接品質與一致性，測試器驗證通斷與對絞，避免隱性故障與重工。
- A詳: 雙絞線對噪音敏感，端接品質直接影響吞吐與穩定。使用壓線鉗、110打線工具、剝線刀能確保導體露銅適當、對絞靠近端口、外皮固定穩固。線測試器可驗證八芯對應、對絞、短路/開路與交叉，及早發現問題，避免完工後難以返工。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q13, C-Q5, D-Q1

A-Q21: 什麼是T568A與T568B？
- A簡: 兩種RJ45打線色序標準，功能等效但須兩端一致；常見家用多採T568B。
- A詳: T568A/B是RJ45端接色序標準，兩者在電氣特性等效。關鍵在於整條通道兩端必須一致，以避免交叉或串音惡化。許多地區常用T568B，辦公或政府專案則可能指定T568A。無論選哪一種，務必標示清楚並全屋一致，便於維護與測試。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q4, C-Q5, D-Q1

A-Q22: 什麼是配線架（Patch Panel）？
- A簡: 配線架集中端接牆面線路，前端以跳線連到交換器，實現靈活配置與維護。
- A詳: 配線架安裝在機櫃，背面用打線方式端接來自各房的雙絞線，正面提供RJ45端口。網路配置改動時，只需更換前端短跳線至不同交換器埠或VLAN，即可完成邏輯變更，降低反覆端接風險。配線架也便於標籤、測試與理線。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: C-Q5, B-Q9, A-Q16

A-Q23: 乙太網路銅纜的長度限制？
- A簡: 標準建議永久鏈路90m、含跳線總長度不超過100m，超過易衰減與錯誤率上升。
- A詳: 對於CAT5e/6/6A，TIA/EIA建議永久鏈路（牆到配線架）90公尺，加上兩端跳線（各不超過5公尺）總長100公尺。超長或劣質端接會造成回波、衰減與誤碼率上升，影響吞吐。規劃時預留合理路徑、避免不必要繞行，並控制彎折半徑與拉力。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q16, D-Q2, C-Q2

A-Q24: 家庭弱電與強電為何要分離？
- A簡: 避免電磁干擾與安全問題，應分管敷設並保持距離，交叉時以90度通過。
- A詳: 強電（市電）會產生電磁場，近距離平行鋪設會誘發弱電（網路、電話）噪聲與錯誤。建議強弱電分管、保持距離（常見至少15-30公分），必要交叉以垂直90度，並避免與高干擾源（變壓器、馬達、燈具電感）並行。安全與性能可同時兼顧。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q12, C-Q2, D-Q2

A-Q25: 什麼是服務圈（Service Loop）與為何預留？

A-Q26: 為何要標籤與編號？

A-Q27: 為何二手設備常是成本效益選擇？

A-Q29: ISP進線與家庭內部分配的關係？

### Q&A 類別 B: 技術原理類

B-Q1: 家庭結構化佈線如何運作？

B-Q3: 用CAT5e承載電話的機制是什麼？

B-Q4: T568B端接的線序與注意事項？

- A詳: 原理說明：T568B定義了雙絞線在RJ45的針腳分配，確保差分對維持與串音控制。步驟：1) 剝外皮保留絞距；2) 按色序入槽；3) 打線刀壓接；4) 固定外皮於拉力槽；5) 測試。核心組件：Keystone或配線架、110打線工具、線測試器。

### Q&A 類別 C: 實作應用類（10題）

C-Q1: 如何規劃每房間的網路孔與電話孔數量？

- A詳: 具體實作步驟：1) 列出每房設備（PC、NAS、印表機、機上盒、遊戲機）；2) 依需求配置至少2xRJ45+1xRJ11；3) 關鍵房（客廳、書房）可到4-6孔；4) 以CAT5e/6拉線，電話孔用RJ45打線保留彈性。關鍵設定：建立面板編碼。注意事項：避免依賴外露小交換器；預留空管。最佳實踐：需求+1原則，多留一孔防未來。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q8, A-Q17, C-Q2

C-Q2: 如何在裝潢期間完成星狀佈線？
- A簡: 設計中心配線間，從每點拉獨立線回機櫃，控長度、分管敷設，完工前全數測試。
- A詳: 具體實作步驟：1) 選配線間/機櫃位置；2) 規劃線路與管徑；3) 從面板盒穿管到機櫃；4) 拉CAT5e/CATV/電話至中心；5) 端接面板與配線架；6) 標籤與測試。關鍵設定：永久鏈路<90m。注意事項：弱強電分離、彎曲半徑≥4倍線徑。最佳實踐：每端預留服務圈。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q2, A-Q3, A-Q23

C-Q3: 如何用CAT5e拉電話並保留改網路孔的彈性？
- A簡: 以RJ45端接T568B，電話暫用轉接線；日後改網路只需更換面板或跳線配置。
- A詳: 具體實作步驟：1) CAT5e照T568B端接於RJ45面板；2) 以RJ45-to-RJ11轉接線供電話；3) 機櫃端連PBX或外線；4) 未來改網路改接交換器即可。關鍵設定：標記此孔位可雙用。注意事項：避免分兩裝置共用一條（split）導致串音。最佳實踐：保留第三孔作純網路。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: A-Q6, B-Q3, D-Q5

C-Q4: 如何選購與安裝15U小機櫃？
- A簡: 選深度>設備長度、良好通風與承重；預留前後維護空間，搭配PDU與理線器。
- A詳: 具體實作步驟：1) 量測設備深度（交換器、DVR/NAS）；2) 選擇通風門與風扇位；3) 安裝PDU/UPS；4) 固定於雜物間靠牆；5) 安裝配線架、理線器。關鍵設定：前後各≥50cm維修距。注意事項：隔離濕熱源、控噪。最佳實踐：門鎖與溫度監控。
- 難度: 中級
- 學習階段: 核心

C-Q5: 如何安裝配線架與面板並正確壓線？
- A簡: 依T568B端接，保持對絞、固定外皮；完成後用測試器驗證八芯與長度。
- A詳: 具體實作步驟：1) 剝外皮≤25mm；2) 依色序入槽；3) 用110打線工具壓切；4) 固定外皮於拉力槽；5) 面板端同法；6) 測試通斷與對序。關鍵程式碼/設定：無，重端接規範即設定。注意事項：避免超彎與拉伸。最佳實踐：加裝理線條與服務圈。

- 學習階段: 核心

C-Q6: 如何集中CATV與電話到機櫃並配置？

- A詳: 具體實作步驟：1) 同軸線拉回機櫃；2) 安裝分配器與強波器；3) 電話線回PBX/外線分配；4) 面板/配線架標籤；5) 測試每路訊號。關鍵設定：分配損耗與增益匹配。注意事項：CATV接地與回傳通道。最佳實踐：將分配器、強波器固定於機櫃側軌。

C-Q7: 如何安裝24埠交換器與做基本網路設定？

- A詳: 具體實作步驟：1) 交換器上架；2) 上行連路由LAN；3) 用短跳線連配線架端口；4) 設管理IP與VLAN。關鍵設定（示例）：管理IP 192.168.10.2/24，VLAN10主網、VLAN20 IoT。注意事項：理線避免擠壓；韌體更新。最佳實踐：DHCP保留NAS/印表機IP。

C-Q8: 如何部署家庭監控錄影與網路連線？

- A詳: 具體實作步驟：1) 攝影機佈點與布線；2) PoE交換器或注入器供電；3) NVR連網設IP/時間；4) 加入攝影機串流；5) 設定錄影與事件。關鍵設定：RTSP/ONVIF啟用，NTP校時。注意事項：外網存取採VPN或雲中繼強化安全。最佳實踐：關鍵鏡頭與NVR接UPS。

C-Q9: 如何建立標籤編碼與文件管理？

- A詳: 具體實作步驟：1) 定義命名（如F2-RmA-P01-Port1）；2) 產生配線對映表；3) 面板/配線架貼標；4) 版本化文件。關鍵「設定」：用試算表維護端口-設備-IP-VLAN。注意事項：變更即時更新；備份雲端。最佳實踐：使用熱縮管與色環區分用途。

C-Q10: 如何做預算控管，兼顧成本與效益？

- A詳: 具體實作步驟：1) 列項（線材、面板、配線架、機櫃、交換器、工具、測試器）；2) 以CAT5e為主、關鍵幹線評估CAT6A；3) 設備優先二手交換器/機櫃；4) 預留孔位與空管。關鍵設定：總預算配置60%材料、30%設備、10%工具。注意事項：不要省端接與測試。最佳實踐：二手買回壓力測試。

### Q&A 類別 D: 問題解決類（10題）

D-Q1: 壓線後測不通怎麼辦？
- A簡: 先做線序與通斷測試，檢查對絞與外皮固定，必要時重打線或更換插座模組。
- A詳: 問題症狀：測試器顯示開路/短路/線序錯。可能原因：色序錯、外皮未固定、導體未切齊、對絞鬆散。解決步驟：1) 拍照比對色序；2) 重打線；3) 更換疑似不良Keystone；4) 再測長度與對應。預防措施：端接時保持對絞、使用優質工具與測試器。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q4, B-Q13, C-Q5

D-Q2: 家用網路效能不佳的常見原因？
- A簡: 端接不良、線過長、干擾、老舊設備或配置不當；逐一排查線材、設備與設定。
- A詳: 症狀：延遲高、掉速、頻繁斷線。可能原因：線長>標準、NEXT/RL劣、強弱電並行、分線（split pair）、老舊Hub、路由性能不足。解決步驟：線測試、換短跳線、重端接、隔離干擾、升級交換器/路由。預防：遵守長度限制、使用UTP與分離管線、定期韌體更新。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q23, B-Q12, B-Q16

D-Q3: CATV出現雪花或馬賽克怎麼辦？
- A簡: 檢查F頭與分配器、量測訊號強度與回傳，必要時調整強波器與分配架構。
- A詳: 症狀：影像雪花、數位方塊化、頻道遺失。可能原因：接頭鬆動、分配損耗過大、回傳阻塞、強波器設置不當。解決步驟：重壓F頭、簡化分配鏈、調整增益、聯繫ISP測信。預防：星狀佈線、用優質同軸與接頭、確保接地。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q5, C-Q6, B-Q19

D-Q4: 電話有雜音或回音如何處理？
- A簡: 確認端接與分配，減少並聯，檢查PBX設定與接地，必要時更換線材或濾波器。
- A詳: 症狀：通話雜音、回音或斷續。原因：接觸不良、過長並聯、PBX增益不當、外線品質差。解決：重端接、改星狀接至PBX、調整音量參數、測外線、加裝濾波器。預防：用CAT5e端接、減少串接、線路分離。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q6, C-Q6, A-Q6

D-Q5: 能否用一條雙絞線分作兩個網路孔？
- A簡: 不建議。分線會破壞對絞與標準，造成串音與降速，應增拉線或用小型交換器。
- A詳: 症狀：偶發斷線、速度降至百兆、封包遺失。原因：分線使用非對應對，破壞差分傳輸與抗干擾。解決：恢復單線單孔，或在該點加小型交換器。預防：規劃足夠孔位與預留，避免非標準接法。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: C-Q3, A-Q17, A-Q23

D-Q6: 機櫃過熱怎麼辦？
- A簡: 優化進出風道、增設風扇與理線、降噪替換風扇、加UPS監控溫度並清潔濾網。
- A詳: 症狀：設備溫度高、降速或自動關機。原因：通風不良、灰塵阻塞、風扇老化、密度過高。解決：裝前後門風扇、整理線束不擋風、清潔濾網、移除不必要設備。預防：規劃散熱、預留空間、定期保養，採二手設備需優先檢測風扇與電源。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q10, A-Q28, C-Q4

D-Q7: 交換器燈閃但無法上網如何診斷？
- A簡: 檢查路由器WAN/LAN、DHCP、IP配置與VLAN；確認上行線與PPPoE或光纖終端狀態。
- A詳: 症狀：區網可通但無法上網。原因：路由器斷線、WAN設定錯誤、DHCP關閉、VLAN隔離、ISP設備故障。解決：重啟數據機/路由、確認PPPoE/靜態IP、檢查上行跳線與VLAN。預防：文件化設定、標籤上行埠、UPS保護。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q14, C-Q7, A-Q29

D-Q8: 監控畫面無法遠端觀看怎麼辦？
- A簡: 檢查NVR與攝影機IP、時間與網路；透過VPN或雲中繼取代DMZ/開埠，強化安全與可達性。
- A詳: 症狀：外網無法連入或畫面中斷。原因：IP衝突、時間不同步、NAT/開埠錯誤、ISP封鎖。解決：校時NTP、DHCP保留或固定IP、測內網RTSP、改用VPN或廠商雲。預防：使用DDNS+VPN、紀錄設定、UPS保障。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q7, C-Q8, A-Q29

D-Q9: PoE攝影機不供電如何處理？
- A簡: 驗證PoE標準相容、功率需求與線長；更換線材或改用注入器，檢查交換器PSE設定。
- A詳: 症狀：攝影機不亮或斷續。原因：PSE/PD標準不匹配（af/at/bt）、功率不足、線材衰減、端接不良。解決：確認耗電、換到高功率埠、縮短線長、重端接、測試注入器。預防：規劃功率餘裕≥30%、用優質線材與端接。
- 難度: 中級
- 學習階段: 進階
- 關聯概念: B-Q18, A-Q30, C-Q8

D-Q10: 線路太多難管理如何整頓？
- A簡: 建立編碼規則與配線表，使用理線器與色環，整理跳線長度並更新文件。
- A詳: 症狀：機櫃凌亂、調整耗時。原因：缺乏編碼與文檔、跳線過長、無理線。解決：制定命名、重貼標籤、換短跳線、加理線器、同步更新配線表。預防：新增/變更即更新文件、定期盤點，保持面板與配線架對應清晰。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: C-Q9, A-Q26, B-Q9
