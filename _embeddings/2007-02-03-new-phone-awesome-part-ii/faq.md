- A詳: Dopod C720W 是搭載 Windows Mobile 5.0 的智慧型手機，採用 QWERTY 實體鍵盤，適合大量輸入與效率應用。可安裝第三方軟體，涵蓋語音命令、鬧鐘工具、RSS 閱讀、導航、字典等；內建藍牙、支援與電腦透過 ActiveSync 同步，亦可經藍牙分享電腦網路。其擴充性與可客製化能力，正是作者選擇與推薦的核心理由。

Q7: 什麼是 ClearType？為何在手機上有意義？

Q11: 手機上使用藍牙與 Wi‑Fi 的差異是什麼？

Q14: 什麼是硬重置（Hard Reset）？風險為何？

### Q&A 類別 B: 技術原理類

Q1: 語音命令如何運作？背後機制是什麼？
- A簡: 以語音辨識引擎比對指令關鍵詞，觸發撥號與查詢，支援自然語句。
- A詳: 語音命令依賴語音辨識（ASR）引擎，將輸入語音轉為音素序列，與預建語法/詞庫比對，解析出意圖（如呼叫、查詢）。熱鍵啟動錄音，辨識後觸發對應 API（撥號、開啟聯絡人）。因採無訓練模式，需在降噪、回授確認（如「確定」）上保障準確率。系統也利用通訊錄名稱與聯絡方式做語境限定，提升辨識成功率。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q4, C-Q1, D-Q5

Q2: pTravelAlarm 與鍵盤鎖的互動機制為何？為何會鎖死？
- A簡: 鬧鐘喚醒與鍵盤鎖事件衝突，導致解鎖 UI 無法正確接管焦點。
- A詳: 早期版本在鬧鐘響起時喚醒裝置，與系統鍵盤鎖的狀態機衝突。當鬧鐘畫面疊在鎖屏 UI 之上，解鎖按鍵事件被攔截或焦點不在正確控制項，導致無法解鎖。新版修正了喚醒順序與按鍵處理，確保鬧鐘結束後鎖屏能正確恢復。此類問題常見於第三方與系統殼層的事件競態。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q5, C-Q2, D-Q1

Q3: 自動鎖鍵與螢幕保護的設計與流程？
- A簡: 透過閒置計時器監控輸入，逾時觸發鎖鍵並渲染自訂資訊層。
- A詳: 工具監聽輸入與系統活動，啟動閒置計時器；超過門檻即呼叫鎖鍵 API，並以全螢幕層顯示自訂畫面（時鐘、狀態圖示）。其狀態圖示讀取系統服務（藍牙、Wi‑Fi、電池、通話）即時值，週期更新。解鎖按鍵通常透過全局熱鍵或特定組合避免誤觸。設計重點是事件優先序與資源節能。

Q4: ClearType 的字型平滑原理是什麼？
- A簡: 利用 RGB 子像素獨立控制，細化邊緣階調以平滑文字邊界。
- A詳: LCD 每像素包含 RGB 子像素。ClearType 以次像素為單位計算字型邊緣覆蓋比例，調整各色通道亮度，等效於水平解析度提升約三倍，達到更平滑的筆劃。其效果依子像素排列（RGB/BGR）與旋轉方向而異；錯配可能模糊或彩邊。手機小螢幕上改善明顯，但需權衡個人感受與耗電。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q7, C-Q4, D-Q6

Q5: 手機 RSS 同步與「線上閱讀」的流程？
- A簡: 先抓取 feed 摘要存本機，點選再呼叫瀏覽器開啟全文網址。
- A詳: RSS 客戶端根據訂閱清單，透過網路抓取 XML feed，解析標題、摘要與連結並存入本機資料庫。離線瀏覽顯示已同步內容；點「線上閱讀」再交由 IE 啟動 HTTP 連線，使用系統目前的網路路徑（Wi‑Fi、行動數據或 BT+ActiveSync）。此設計兼顧離線效率與線上延展性。
- 難度: 初級
- 學習階段: 核心

Q6: 藍牙 + ActiveSync 上網的原理是什麼？
- A簡: 透過藍牙序列埠連結 PC ActiveSync，使用電腦分享的網際網路。
- A詳: 手機與 PC 透過藍牙序列埠（SPP）配對成虛擬 COM 連線，由 ActiveSync 建立同步會話並提供「網際網路直通」功能，將手機的 HTTP/Socket 請求轉送至 PC 網路介面。相較 Wi‑Fi，SPP 頻寬較低，但對於瀏覽文字與 RSS 已足夠，且連線步驟簡單、耗電較小，適合短時上網。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q11, C-Q6, D-Q3

Q7: Smart Journal 如何將通話紀錄寫入 Outlook？
- A簡: 監聽 ActiveSync 事件，擷取手機通話記錄並轉存到 Outlook 日誌。
- A詳: 外掛註冊為 ActiveSync 同步事件的觀察者，連線建立時觸發擷取程序：讀取手機 CallLog（含來電/去電/未接、聯絡人匹配、通話時長），轉為 Outlook Journal 項目並寫入對應欄位。此流程不改動手機資料，純同步匯入。可靠運作仰賴 ActiveSync 自動啟動與 Outlook 可寫入權限。

Q9: 藍牙覆蓋範圍與天線設計的影響？
- A簡: 發射功率與天線效率決定有效距離，環境遮蔽亦強烈影響穩定。
- A詳: 藍牙距離受裝置等級（功率）、射頻前端、天線增益與人體/牆面遮蔽影響。USB 小型 dongle 天線有限，易受擺位影響；PCMCIA 卡常有較佳天線與放大器，實測更穩定。標稱 200m 通常理想值，室內實距短得多。優化可透過換裝硬體、調整擺放與避開干擾源。

Q11: microSD 容量與檔案系統對相容性的影響？
- A簡: 大容量需 FAT32 支援；裝置/OS 限制可能影響讀寫與穩定。
- A詳: microSD 容量越大越可能需要 FAT32（相對 FAT16），而裝置與 OS 對檔案系統支援程度各異。WM5 多支援 FAT32，但早期韌體可能在高容量或大量小檔案時效能下降。建議以可信品牌、適度容量、定期檢錯與避免拔卡寫入中，以確保穩定性。

Q13: ActiveSync 如何觸發外掛（如 Smart Journal）？

Q15: 螢幕保護顯示「溫度」可能來自哪裡？為何不準？
