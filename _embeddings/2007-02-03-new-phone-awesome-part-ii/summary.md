# 新手機真是讚 (y) - II

## 摘要提示
- 購機動機與冷門機型: 選擇 Dopod C720W 前先用 Microsoft Smartphone SDK 模擬驗證需求，雖討論少但功能符合期待。
- 語音命令實用性: 內建「語音命令」免訓練即可撥號、開聯絡人，操作流暢好用。
- 鬧鐘替代方案: 內建鬧鐘功能貧弱，以 pTravelAlarm 補足多時段與自訂鈴聲需求。
- 自動鎖與螢幕保護: 以德國小工具實現類 PC 的螢幕保護與自動鎖鍵盤，並可客製資訊面板。
- 字典與 ClearType: 改裝第三方字典（英英/英漢），搭配 ClearType 顯示更清晰，打字配合 QWERTY 更順手。
- RSS 閱讀與上網策略: 內建 RSS Reader 搭配每日同步，離線閱讀方便，臨時上網以藍牙+ActiveSync 取代 Wi-Fi。
- 導航相容性: PaPaGo G10 在 320x240 解析度上運作正常，導航無虞。
- 通話記錄整合: PC 端 Smart Journal 自動將手機通話記錄同步到 Outlook 日誌，資料更完整。
- 週邊採購與踩雷: 經歷藍牙耳機疑似假貨、擴充 microSD、改用 PCMCIA 藍牙卡提升收訊與便利。
- 後續配件計畫: 計畫再入手平價 GPS 藍牙接收器，持續擴充使用情境。

## 全文重點
作者以實測與工具驗證為前提入手 Dopod C720W，雖是相對冷門的 Windows Mobile Smartphone，但在多項實用功能上表現亮眼。首先，內建「語音命令」軟體不需事先錄音訓練，能直接以語音撥號、開啟聯絡人並確認操作，顯著提升單手或忙碌時的操作效率。內建鬧鐘功能不足，作者延續舊機習慣改用 pTravelAlarm 以支援多組時段與自訂鈴聲，雖曾遇到鍵盤鎖定後無法解鎖的 bug，所幸新版已修復而得以續用。

為解決直板多鍵易誤觸的困擾，作者安裝一款德製小程式，提供電腦式螢幕保護與自動鎖鍵盤，並可自定畫面內容（如類比時鐘與狀態圖示），動態顯示 Wi-Fi、藍牙、電量、漏接來電等資訊，實用性高。因原廠網站缺乏軟體下載與支援，作者改裝先前使用的第三方字典，結合 ClearType 提升閱讀，搭配 QWERTY 鍵盤查詢效率佳。

對於資訊吸收，作者改觀於手機 RSS Reader 的價值：即便每天只同步一次，也能離線閱讀大量新聞摘要；若想看全文，可即時連網。由於 Wi-Fi 操作繁瑣且無行動上網吃到飽，改採家中電腦的藍牙 dongle 搭配 ActiveSync 共享連線，讓人在床上也能臨時上網。導航方面，PaPaGo G10 在 320x240 畫面上運作正常，解除相容性疑慮。

在資料整合上，PC 端的 Smart Journal 能於每次 ActiveSync 啟動時，自動將手機通話紀錄（含來電、去電、未接、通話時間與聯絡人）匯入 Outlook 日誌，對於通聯追蹤非常實用。配件方面，作者先後添購藍牙耳機（遭遇疑似假貨）、1GB microSD（容量綽綽有餘）、並以 PCMCIA 藍牙卡取代 USB dongle，藉由更強收訊範圍提升臥房內使用便利，雖內部以 USB host 方案實作且使用 VIA 晶片讓人皺眉，但實際使用尚稱穩定。最後計畫再收一顆平價藍牙 GPS 接收器，彌補舊裝置誤用變壓器而損壞的遺憾。

整體而言，作者以「能裝軟體、可客製化」的 Smartphone 優勢，拼裝出符合個人習慣的工作流，從語音操作、鬧鐘、螢幕鎖、字典、RSS、導航到通話記錄，同時透過藍牙生態與記憶卡擴充完善體驗；即便原廠支援不足與配件踩雷，仍透過社群與第三方工具補足缺口，最後不忘以半玩笑半勸敗的語氣，推坑朋友加入行列。

## 段落重點
### 開場與購機策略：冷門機型的充分驗證
作者自承不走開箱照片流，強調以實用度為導向。鎖定 Dopod C720W 後，先用 Microsoft Smartphone SDK 與 Device Emulator 模擬所需功能，確定能滿足需求才入手。雖然市場討論度低，但實測結果顯示其優點符合個人工作流。此段定調全文基調：重視軟體可塑性與前期功課，淡化外觀質感比較。

### 語音命令與鬧鐘：日常操作的剛需補強
內建「語音命令」可直接以自然語音撥號與開聯絡人，免訓練、操控順暢，顯著提升效率。內建鬧鐘功能貧弱，作者改用 pTravelAlarm 支援多時段與 MIDI/MP3/WMA 鈴聲；曾因 WM5 上鎖後鬧鐘觸發導致無法解鎖，只能來電或拔電池解圍，所幸一月更新修復。此段凸顯第三方軟體在 Windows Mobile 生態中的關鍵價值。

### 螢幕保護與自動鎖：避免誤觸與資訊一眼掌握
直板多鍵與側邊觸控條易誤觸，作者裝設德製小程式，提供閒置自動鎖鍵盤與可自訂螢幕保護畫面，並以類比時鐘提升觀感。底部圖示列能即時顯示 Wi-Fi、藍牙、電量、漏接來電等系統狀態，兼顧美觀與資訊密度。溫度顯示疑似不準，但不影響整體實用性，解決攜帶與誤按痛點。

### 字典與 ClearType：閱讀與輸入效率
因原廠網站缺乏軟體下載，作者改用先前於 8390 上的字典工具，安裝英英與英漢版並啟用 ClearType，顯示更平滑清晰。配合 QWERTY 鍵盤，查詢體驗流暢；此段也側寫對廠商售後支援的不滿，反襯第三方資源的重要性。

### RSS 與連線策略：離線為主、臨時上網
雖起初質疑手機 RSS 的必要性，但每日一次同步已足夠累積大量可離線閱讀的新聞摘要；若需看全文再即時連線。因 Wi-Fi 操作繁瑣且無 3G/GPRS 吃到飽，作者改以電腦端藍牙 dongle 配合 ActiveSync 共享網路，人在床上即可臨時上網，成為「窩床看 RSS」的新日常。

### 導航與通話記錄：相容無虞與資料留痕
PaPaGo G10 在 320x240 上運作正常，解除解析度疑慮。PC 端 Smart Journal 每次 ActiveSync 啟動即自動將手機通話記錄匯入 Outlook 日誌，包含來/去電、未接、通話時長與識別聯絡人，讓通聯歷史完整可查，對工作或生活管理皆十分有用。

### 配件採購與經驗：從踩雷到升級
作者接連入手數樣配件：藍牙耳機遇疑似假貨（外觀相似但做工差、裝置 ID 相同，感嘆對岸仿冒力道）；1GB microSD（含轉卡與讀卡器），容量綽綽有餘；改用 PCMCIA 藍牙卡取代 USB dongle，提升收訊範圍與攜帶便利，雖內部以 USB host 方案搭配 VIA 晶片令人卻步，但實測穩定。另計畫入手平價藍牙 GPS 接收器以補舊裝置損壞之缺。

### 收尾與勸敗：軟硬體整合的樂趣
總結 Smartphone 的最大價值在「能裝能調」，透過語音、鬧鐘、鎖定、字典、RSS、導航、通話紀錄與藍牙生態，組合出貼合個人習慣的高效工作流。即使原廠支援貧弱與配件踩雷，只要善用第三方工具與社群經驗，仍能獲得卓越體驗。最後以半玩笑口吻勸敗，展現極客式折騰與樂趣。

## 資訊整理

### 知識架構圖
1. 前置知識：
   - 基本行動裝置概念（Windows Mobile/Smartphone 與功能手機差異、QWERTY 鍵盤操作）
   - 無線連線方式（Bluetooth、Wi‑Fi、ActiveSync 透過 PC 共用網路）
   - 常見行動應用類型（語音命令、鬧鐘、RSS、字典、導航）
   - PC 端整合（Outlook 日誌、驅動/配對概念、SDK/模擬器基本使用）

2. 核心概念：
   - 功能可擴充：Windows Mobile 可安裝第三方軟體補齊原生不足（鬧鐘、自動鎖鍵、字典、RSS、PC 同步工具）
   - 多通道連線策略：以 Bluetooth + ActiveSync 取代 Wi‑Fi/行動網路臨時上網
   - 使用者體驗優化：語音命令免訓練、螢幕保護+自動鎖鍵、ClearType 提升可讀性、首頁 RSS 輪播
   - 裝置整合與資料留痕：Smart Journal 將通話紀錄匯入 Outlook 形成完整通聯歷史
   - 週邊與擴充儲存：microSD、藍牙耳機、藍牙 GPS、PCMCIA BT 卡提升收訊範圍

3. 技術依賴：
   - Smartphone 應用運行於 Windows Mobile 5（WM5）之上
   - 語音命令/第三方工具依賴系統 API；ClearType 為系統字型渲染
   - pTravelAlarm 等需要與系統鎖鍵機制相容（曾有鎖鍵後鬧鐘喚醒的相容性 bug）
   - RSS Reader 需透過網路連線（Wi‑Fi 或 Bluetooth+ActiveSync 的 PC 網路共享）
   - Smart Journal 安裝於 PC，依賴 ActiveSync 與 Outlook 日誌
   - PaPaGo G10 導航依賴藍牙 GPS 接收器與裝置解析度（320x240 實測可用）

4. 應用場景：
   - 單手快速操作與撥號（語音命令）
   - 規律鬧鐘/提醒（上班與週末時段區隔）
   - 口袋離線資訊閱讀（RSS 每日同步一次後離線看）
   - 導航出行（PaPaGo + BT GPS）
   - 通聯紀錄與工作追溯（Outlook 日誌集中管理）
   - 在家中床邊/室內行動上網（Bluetooth 取代起身連 Wi‑Fi）

### 學習路徑建議
1. 入門者路徑：
   - 認識 Windows Mobile 介面與基本設定（鍵盤鎖、網路連線、ActiveSync）
   - 安裝並設定語音命令、RSS Reader、字典工具（體驗核心使用情境）
   - 配對 Bluetooth 與 PC，測試經由 ActiveSync 上網

2. 進階者路徑：
   - 使用自動鎖鍵與可客製螢幕保護（監看裝置狀態圖示）
   - 優化鬧鐘工具（pTravelAlarm）與檢視版本修正相容性
   - 將 Smart Journal 部署到 PC，串接 Outlook 日誌，建立通聯檔案化流程

3. 實戰路徑：
   - 導航整合：PaPaGo G10 + 藍牙 GPS 實場測試（路線、電源、收訊）
   - RSS 工作流：訂閱來源管理、每日同步策略、線上全文跳轉
   - 週邊選購與佈署：挑選穩定 BT 配件、microSD 儲存規劃、PCMCIA/USB BT 取捨與收訊測試
