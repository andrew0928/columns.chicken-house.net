# 水電工日誌 6. 機櫃設備展示

## 問題與答案 (FAQ)

### Q&A 類別 A: 概念理解類

Q1: 什麼是機櫃（Server Rack）？
- A簡: 機櫃是依U高標準化的設備架，集中安裝伺服器、網通與配電，提供固定、散熱、布線與維護的統一空間。
- A詳: 機櫃是一種依「U」為高度單位的標準設備架，常見有門板、側板、導軌、層板、風扇與內建排插等。它能集中安裝伺服器、交換器、配線架與監控等，讓供電、散熱、布線與維護更有秩序。文中採用15U機櫃，具透明壓克力前門、可拆後門、輪子與支撐腳、底部防蟲線孔、上方風扇與內建排插，適合家用小機房整合。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q2, B-Q1, C-Q2

Q2: 什麼是U（Rack Unit）？一個U代表多高？
- A簡: U是機櫃高度單位，1U為1.75英吋（約44.45mm），用以衡量設備與機櫃可用空間。
- A詳: U（Rack Unit）為機櫃高度的標準單位，1U=1.75英吋≈44.45毫米。機殼、配電與網通設備皆依此設計，如4U機殼高度約4×1.75英吋。選擇機櫃時，需估算設備總U高與餘裕空間，以便散熱、線材轉彎半徑與未來擴充。文中選用15U機櫃，剛好容納伺服器、交換器、配線架與DVR等。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q1, A-Q4, C-Q2

Q3: 為什麼選擇15U而不是更小的機櫃？
- A簡: 15U在小空間中兼顧現況與擴充，設備剛好容納且保留餘裕，避免日後擴充受限。
- A詳: 機櫃尺寸需同時滿足「現況可容納」與「未來可擴充」。過小會導致設備擁擠、散熱與布線困難；過大則浪費空間與成本。文中以15U替代原先想買的小尺寸，實際擺放後「剛剛好」，並保留層板與空位，利於增加硬碟、交換器或監控設備。這種兼顧與餘裕的規劃，能降低後續更換機櫃的成本與風險。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q1, C-Q2, B-Q1

Q4: 什麼是4U機架式機殼？有何特點與用途？
- A簡: 4U機殼符合機櫃安裝高度，能裝ATX主機板與多顆硬碟，空間大、散熱與維修性佳。
- A詳: 4U機殼是高度為4U的機架式機殼，通常支援ATX主機板與多顆硬碟，適合家用伺服器或NAS應用。文中4U機殼內含8cm風扇與濾網、四顆HDD與光碟機，兼顧散熱與防塵，讓硬碟溫度維持在40°C以下。其缺點是重量較重，上架需注意安全；優點是空間大、擴充性與可維護性佳，很適合多服務合一的家庭伺服器。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q8, C-Q3, D-Q10

Q5: 什麼是配線架（Patch Panel）？與交換器有何差異？
- A簡: 配線架是終端與整理網路線的被動面板，交換器則是主動轉發封包的網路設備。
- A詳: 配線架屬被動設備，將來自各房間或壁插的網路線集中終端，便於標示、測試與重新跳接；其背面以打線方式固定，前面用跳線連到交換器。交換器則是主動設備，學習MAC位址並轉發乙太封包。兩者常搭配：配線架負責佈線管理，交換器負責資料轉發。文中配線架位於交換器下方，方便集中管理。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q7, B-Q9, C-Q6

Q6: 什麼是NAT？為何家用網路需要它？
- A簡: NAT將內網私有IP轉換為外網可達位址，讓多台裝置共用一條對外連線。
- A詳: NAT（Network Address Translation）可在出口設備（如路由器或伺服器）將多台內網裝置的私有IP，轉換為單一或少數外部IP，共用上網。它也提供基本連線追蹤與封包過濾，增進安全性。文中伺服器負責家中NAT並承載Web、DNS、VPN等服務，將內外網資源整合，適合家用與小型辦公環境。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q2, C-Q4, D-Q5

Q7: 什麼是RAID1（鏡像）？為何用於共享資料夾？
- A簡: RAID1以兩顆硬碟互為鏡像，提升資料冗餘與可用性，適合重要檔案共享。
- A詳: RAID1將資料同時寫入兩顆硬碟，任一顆損壞仍可持續運作並快速復原，著重可靠度而非容量或效能。文中以RAID1承載家用共享資料夾，平衡成本與資料安全。其讀取可分散，寫入需同步兩碟；需定期監控狀態並準備備援硬碟，以縮短故障重建時間。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q3, C-Q5, D-Q4

Q8: 什麼是電話總機（PBX）？家用為何需要？
- A簡: PBX集中管理多條外線與多部分機，提供轉接、保留等功能，解決多電話佔桌與雜音。
- A詳: PBX（Private Branch Exchange）能將數條外線分配給多部分機，並提供轉接、代接、保留、內線互打與線路選擇等。文中因家中有多條外線且話機過多、雜音與供電不足問題，改以簡易PBX整合。其採用一般話機，成本低但功能以撥碼指令操作，較少燈號。對家用與SOHO而言，是整齊、可靠的話務解決方案。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q4, C-Q7, D-Q2

Q9: 專用PBX話機與普通話機有何差異？
- A簡: 專用話機具燈號與快捷鍵，操作直覺但較貴；普通話機便宜普及，需以指令撥碼操作。
- A詳: 專用話機可顯示多線狀態、保留與轉接燈號，提供一鍵操作，學習成本低但價格高且綁品牌。普通話機價格低、選擇多，但功能靠撥碼，如「#11選外線」「##保留/掛斷」等，需記指令。文中採用普通話機的PBX，兼顧成本與可用性，適合家用場景。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: A-Q8, C-Q7, D-Q2

Q10: RJ11與RJ45有何差異與用途？
- A簡: RJ11常用於電話線路，RJ45用於乙太網路。兩者尺寸與針腳不同，不可互用。
- A詳: RJ11通常為2～6芯接頭，應用於類比電話、PBX外線/內線；RJ45為8芯接頭，主要用於乙太網路（Cat5e/Cat6）。配線架與交換器採RJ45，PBX背板多為RJ11。正確辨識與佈線能避免物理損壞與訊號不良，提升系統穩定性。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q6, A-Q8, B-Q7

Q11: 什麼是監控錄影主機（DVR）？與PC錄影差別？
- A簡: DVR是專用錄影設備，穩定省心；PC錄影彈性高但易受軟硬體干擾、維護負擔大。
- A詳: DVR將影像擷取、壓縮、儲存與回放整合於專用硬體，設定簡單、長期運作穩定。PC錄影雖擴充彈性高，卻常受驅動、OS更新與共用用途影響，穩定性與維護成本偏高。文中由PC錄影轉為DVR，配置160GB硬碟，並啟用動態偵測錄影，有效降低容量消耗。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q5, C-Q8, D-Q3

Q12: 什麼是動態偵測錄影？有何優點？
- A簡: 依畫面變化自動啟停錄影，節省儲存空間並聚焦重要片段，適合長時間監控。
- A詳: 動態偵測錄影以影格差異或區域門檻判斷是否有「動作」，有變化才錄影並可設定前/後錄秒數。其優點是顯著節省磁碟空間、降低檢索負擔，並提高事件關聯性。文中以160GB就能錄數日，即是其效率展現。需留意靈敏度設定，避免誤報或漏錄。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q5, C-Q8, D-Q4

Q13: 什麼是Java Applet與JRE？為何DVR需安裝？
- A簡: 部分DVR的Web檢視以Java Applet實作，瀏覽需JRE支援，否則畫面無法載入。
- A詳: Java Applet為在瀏覽器中執行的小程式，需JRE（Java Runtime Environment）提供執行環境。文中DVR具Web介面，若未安裝JRE將無法顯示影像或操作控制。雖然現今瀏覽器對外掛支援趨緊，仍可使用廠商客戶端或相容瀏覽器作為替代方案，確保遠端存取可用。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q6, C-Q8, D-Q6

Q14: 什麼是ADSL數據機？在架構中扮演什麼角色？
- A簡: ADSL數據機將電話線的寬頻訊號轉為乙太網路，連接內網路由/NAT設備上網。
- A詳: ADSL（非對稱數位用戶迴路）透過既有電話線提供寬頻連線。數據機負責調變/解調，將線路訊號轉為乙太網路，供路由/NAT設備使用。文中ADSL數據機上方接外線，下方接交換器與NAT伺服器，構成家用網路的對外出入口。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q6, B-Q2, D-Q9

Q15: 交換器（Switch）與集線器（Hub）有何差異？
- A簡: 交換器依MAC學習按埠轉發，效率高；集線器全埠廣播，易碰撞、效能差。
- A詳: 集線器屬物理層裝置，將訊號放大後廣播至所有埠，分享同一碰撞域。交換器為資料鏈路層裝置，維護MAC表，僅將封包轉發到目的埠，並隔離碰撞域。文中採用24埠交換器取代舊式Hub，提升穩定性與吞吐；GBE升級可視流量需求規劃。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q9, C-Q9, D-Q8

Q16: 為何拔掉機櫃後門並關閉機頂風扇？
- A簡: 降低噪音並改善被動對流；在負載與溫度允許下，犧牲部分正壓換靜音。
- A詳: 家用環境對噪音敏感，若設備熱密度低，開放式氣流路徑能以自然對流與設備自帶風扇達成散熱。文中因機頂風扇過吵而停用，並拆後門減少風阻，兼顧靜音與可接受溫度。需持續監測溫度，視情況調整風扇轉速或增設靜音風扇。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q1, C-Q10, D-Q1

Q17: 什麼是防蟲線孔？為何重要？
- A簡: 防蟲線孔允許線材穿越又阻隔害蟲，避免蟑螂螞蟻入侵造成髒汙與短路風險。
- A詳: 防蟲線孔通常加裝刷條或密封圈，兼顧走線彈性與阻隔縫隙。家用機房易與雜物共存，害蟲入侵會帶來灰塵、排泄物與短路風險。文中由賣家預先開孔，可維持整潔與可靠度。實務上應搭配濾網、定期清潔，降低積塵與過熱風險。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q8, C-Q2, D-Q4

Q18: 為何要鎖上機櫃門？
- A簡: 防止孩童誤觸與設備滑落，保護重物上架時的安全與設備穩定運作。
- A詳: 機櫃內含重物與高壓電，孩童或非授權人員誤觸易受傷或造成停機。文中4U主機重量逾十公斤，上下架風險高，因此上鎖可避免推拉門時意外。良好做法含門禁管理、上架導軌、雙人作業與標示警告，全面提升安全。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: D-Q10, C-Q2, C-Q3

Q19: 機房擺「乖乖」的由來與含義是什麼？
- A簡: 一種IT圈迷因，象徵祈求系統穩定；常選綠色包裝寓意「一切正常」。
- A詳: 在台灣IT圈，相傳在機房或伺服器旁放「乖乖」零食能帶來系統穩定。綠色包裝象徵正常、紅色象警告的趣味聯想。本文以幽默態度擺放綠色「椰子口味」乖乖，並分享「吃了會當機」的玩笑。雖無技術效益，卻反映工程文化的團隊默契與儀式感。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: D-Q1, C-Q1, A-Q16

Q20: 為何選用二手或平價設備？
- A簡: 降低成本、滿足家用負載，先求可用與穩定；日後再依需求升級。
- A詳: 家用與SOHO環境多以成本效益優先，挑選二手或入門型設備即可滿足當前頻寬與服務需求。文中採二手24埠交換器與平價PBX，先建立可用架構，再視GBE或進階功能逐步升級。關鍵在於規劃餘裕、留擴充位與可替換性，避免一次到頂卻造成浪費。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q9, D-Q8, A-Q3


### Q&A 類別 B: 技術原理類

Q1: 機櫃散熱與氣流如何運作？
- A簡: 以前進後出或下進上出的氣流路徑，配合風扇、過濾與開孔，兼顧溫度與噪音。
- A詳: 核心原理是建立低阻力氣流路徑與足夠換氣量。常見為前進後出、下進上出。風扇提供動壓，濾網濾塵但增阻。門板、側板與開孔決定阻力與對流效率。家用低熱密度時，可透過拆後門、停用噪音風扇，依設備自帶風扇與自然對流散熱；但需持續監控CPU、HDD與機櫃內溫，必要時以低轉速風扇或風扇控制器補強。核心組件：風扇、濾網、門板/開孔、溫度監控。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q16, B-Q8, C-Q10

Q2: NAT的運作機制是什麼？
- A簡: 透過連線追蹤將內網多IP映射到外網位址/埠，回應時依表查找還原轉送。
- A詳: NAT/NPAT以連線追蹤表（五元組）記錄內外端映射。內網主機外連時，出口裝置改寫來源IP/Port並建立表項；外部回應抵達後依表還原並轉回內機。對外服務需設定Port Forward。流程：啟用轉發→建立SNAT/MASQ→必要時DNAT。核心組件：連線追蹤、SNAT/DNAT規則、狀態表與逾時機制。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q6, C-Q4, D-Q5

Q3: RAID1的讀寫與容錯原理為何？
- A簡: 寫入時同時寫兩碟，讀取可分散；單碟故障仍可運作並以健康碟重建。
- A詳: RAID1將資料鏡像到兩顆硬碟。寫入需雙寫，延遲取決於較慢一顆；讀取可選最快碟或交錯讀。單碟失效時陣列降級但可運行，換新碟後依位元組重建。需監控SMART與陣列狀態，避免雙碟同時故障。核心組件：陣列超區（metadata）、鏡像邏輯、重建流程與監控告警。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q7, C-Q5, D-Q4

Q4: 類比PBX的交換與撥碼控制機制是什麼？
- A簡: 以交換矩陣將分機與外線接通，透過DTMF指令觸發選線、轉接與保留等功能。
- A詳: 類比PBX內含CO外線介面、分機介面與交換矩陣。使用者拿起話機送DTMF（按鍵音），PBX解析碼元，如「#11選外線1」「##保留/掛斷」，據以建立/釋放路徑並控制鈴流。外線使用狀態以指示燈顯示。核心組件：電源/鈴流、DTMF偵測、交換矩陣、號碼規則與權限表。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q8, A-Q9, C-Q7

Q5: DVR的動態偵測與錄影流程如何運作？
- A簡: 以影格差分與門檻判斷動作，觸發前後錄，循環儲存以最佳化容量。

Q8: 硬碟散熱與濾網的工作原理是什麼？

### Q&A 類別 D: 問題解決類

Q1: 機頂風扇太吵怎麼辦？
- A簡: 降速、換靜音風扇、優化氣流或停用非必要風扇，並持續監測溫度。
- A詳: 症狀：高頻噪音、共振或風切聲。原因：高速風扇、阻力大、安裝鬆動。解法：1) 降轉或換低噪風扇。2) 減少阻力（整理線材、拆後門）。3) 墊片防震。4) 非必要時停用，改靠自然對流。預防：選靜音規格、良好固定、定期除塵與溫度監控。

- A詳: 症狀：雜音、鈴聲微弱或無法撥號。原因：REN總負載過高、線路品質差。解法：1) 以PBX集中接外線，再分配分機。2) 減少並聯或改用低REN話機。3) 必要時加振鈴放大器。預防：良好佈線與端接、REN預估、接地與防雷。

Q9: ADSL網路常斷線該怎麼診斷？
- A簡: 檢查線路雜訊與分離器、縮短電話線、穩壓供電與韌體更新，必要時報修。
- A詳: 症狀：頻繁斷線、速率不穩。原因：線路雜訊、分離器故障、佈線過長、電源不穩。解法：1) 更換分離器與線材。2) 將數據機靠近入口。3) 穩壓與降溫。4) 檢視SNR/Line Attenuation。5) 向ISP報修。預防：妥善佈線、過濾器品質與定期檢視記錄。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q14, C-Q1, A-Q8

Q10: 重物上架與孩童安全如何保障？
- A簡: 採雙人上架與導軌、支撐腳固定與門禁上鎖，標示警告並分層承重。
- A詳: 症狀：上架滑落、夾手、傾倒風險。原因：單人作業、未固定、超載。解法：1) 雙人作業與使用導軌/托盤。2) 調整支撐腳、固定防傾。3) 上鎖與標示。4) 重心下移，重下輕上。預防：上架SOP、承重驗證與定期檢查螺絲。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q2, A-Q18, A-Q4
