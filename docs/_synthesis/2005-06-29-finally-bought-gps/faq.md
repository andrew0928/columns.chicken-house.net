# 終於買了 GPS

## 問題與答案 (FAQ)

### Q&A 類別 A: 概念理解類

A-Q1: 什麼是 GPS？
- A簡: GPS是全球衛星定位系統，透過接收多顆衛星訊號，計算使用者位置、速度與時間。
- A詳: GPS（Global Positioning System）是一套由多顆導航衛星、地面控制站與使用者接收機組成的定位系統。接收機同時接收至少四顆衛星的時間與軌道資訊，利用距離差計算出經緯度與高度，並提供速度與時間資訊。它是導航軟體定位來源的核心，能在地圖上標示當前位置，支援路徑導航、軌跡記錄與定位服務。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q1, A-Q14, A-Q18, B-Q8

A-Q2: 什麼是藍牙 GPS 接收器？
- A簡: 以藍牙傳輸定位資料的外接GPS裝置，供PDA/手機讀取NMEA資料。
- A詳: 藍牙GPS接收器是一種獨立硬體，負責接收衛星訊號並以藍牙（SPP）對外輸出標準NMEA資料流。當PDA或手機內無內建GNSS時，可透過配對建立虛擬COM埠，讓導航軟體讀入定位資訊。其優點是相容性高、可更換、續航獨立，常見於早期PDA情境。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q2, C-Q1, C-Q9, A-Q15

A-Q3: 什麼是 PDA？在導航中扮演什麼角色？
- A簡: PDA是可攜式數位助理，作為導航軟體平台與人機介面。
- A詳: PDA（Personal Digital Assistant）是早期的掌上型電腦，具備運算、顯示與藍牙等通訊功能。於導航情境中，PDA負責執行導航軟體、展示地圖與提示路徑，並透過藍牙讀取外接GPS資料。它是整體方案的「軟體與介面載體」，處理規劃、語音提示與使用者操作。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q16, C-Q2, D-Q8

A-Q4: 什麼是導航軟體？Papago 屬於哪一類？
- A簡: 導航軟體以地圖與定位規劃路徑并提供語音指引；Papago屬車用導航。
- A詳: 導航軟體將GPS定位與數位地圖結合，提供搜尋目的地、路徑規劃、語音提示、即時位置顯示等功能。Papago屬於車用/行動端離線導航軟體，常見於PDA與早期智慧手機環境。它讀取NMEA定位資料，於地圖上進行匹配、導航與重新規劃。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q3, C-Q4, D-Q3

A-Q5: 為什麼 GPS 對路痴是救星？
- A簡: 位置即時可視化，減少迷路與焦慮，能迅速回到認得路段。
- A詳: 對路痴而言，最困難是搞清楚「我在哪」。GPS能持續在地圖上顯示當前位置、方向與路名，讓人即使在陌生區域也能校準方向。即使路徑規劃不完美，位置顯示仍可幫助找回熟悉道路與地標，降低錯路成本與決策壓力，提升探索與移動自信。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q6, A-Q7, C-Q8

A-Q6: 為何即使路徑規劃不佳，顯示當前位置仍很有用？
- A簡: 位置是基礎事實，可自助判斷路況，迅速修正決策。
- A詳: 路徑規劃屬建議，不一定最優；但「我在哪」是關鍵事實。有了即時位置，使用者可比對環境、路標與地圖，自行選擇更直覺的替代路線；即使偏離路徑，仍能沿可見道路前進並重回主幹線。故定位可大幅降低規劃瑕疵的負面影響。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, A-Q7, B-Q4, D-Q5

A-Q7: 定位（Positioning）與導航（Routing）有何差異？
- A簡: 定位找「在哪」；導航算「怎麼走」，兩者相輔相成。
- A詳: 定位是利用衛星與感測器求得使用者當前位置、速度與方向；導航則是基於位置與地圖資料，計算從起點到目的地的路徑並提示轉向。定位是輸入，導航是決策輸出；定位越穩定，導航越可靠；導航算法良好，能更好地利用定位結果。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q1, B-Q3, A-Q6

A-Q8: 藍牙外接 GPS 與內建 GPS/手機 GNSS 有何差異？
- A簡: 外接可替換與獨立續航；內建整合好、攜帶方便。
- A詳: 外接藍牙GPS優點在於可更換、靈敏度高、對舊裝置友善，並有獨立電池；缺點是需配對、攜帶多一件。內建GNSS整合佳、啟用簡易、耗電由主機管理，但不可單獨升級與更換。選擇取決於裝置年代、使用頻率與攜帶習慣。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q9, D-Q2, A-Q15

A-Q9: 什麼是數位地圖？離線與線上差別？
- A簡: 數位地圖以向量/點陣呈現道路；離線穩定，線上即時性佳。
- A詳: 數位地圖包含道路網、限速、單行、POI等資料。離線圖儲存在本機，無網也可用、反應快；線上圖可即時更新與整合路況，但依賴網路。車用PDA年代多採離線向量圖，適合長途或收訊不佳場景，更新需手動。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q11, C-Q4, C-Q8

A-Q10: 什麼是路徑規劃與導航 AI？
- A簡: 以圖論與啟發式演算法計算最適路徑與轉向提示。
- A詳: 路徑規劃以道路為圖的節點與邊，根據權重（距離、時間、費用）用Dijkstra或A*等演算法求解。導航AI則包含地圖匹配、偏離偵測、重新規劃與語音提示決策。其品質受地圖準確度、偏好設定與實時資訊影響。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q3, D-Q5, B-Q5

A-Q11: 什麼是地圖匹配（Map Matching）？
- A簡: 將雜訊位置對齊道路網，幫助正確顯示與導航判斷。
- A詳: 地圖匹配會把GPS輸出的帶誤差座標，結合道路拓撲、車速與方向，推論最可能所在車道或道路，修正位置跳動與偏移。其可改善城市峽谷或多路徑下的不穩定顯示，提升轉向提示的時機與準確性。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q4, D-Q7, A-Q18

A-Q12: 什麼是 NMEA？為何常見於 GPS？
- A簡: NMEA是GPS資料標準格式，透過序列介面傳遞定位訊息。
- A詳: NMEA 0183是GPS接收機常用輸出協定，採ASCII文字句子（如GGA、RMC）表達時間、位置、速度等。藍牙GPS經SPP虛擬COM輸出NMEA，PDA上的導航軟體讀取解析，即可獲取定位資料。其通用性高、相容性佳。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q14, C-Q2, D-Q3

A-Q13: 什麼是 TTFF（首次定位時間）？
- A簡: 從開機到取得有效定位所需時間，受星曆與環境影響。
- A詳: TTFF描述接收機從啟動到計算出可靠位置的時間。冷啟動需下載星曆/曆書，最久；暖/熱啟用時已有資訊，較快。遮蔽物、電離層與天線敏感度也影響TTFF。車用場景建議開車前先開機預熱，縮短等待。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q7, D-Q1, C-Q5

A-Q14: GPS 與 GNSS 的關係與差異？
- A簡: GPS是GNSS之一；GNSS包含多套衛星系統如GLONASS、Galileo等。
- A詳: GNSS（全球導航衛星系統）是泛稱；美國GPS、俄羅斯GLONASS、歐洲Galileo、中國北斗等皆屬GNSS。現代接收機常多星系多頻，提高可見衛星數與抗遮蔽能力。早期藍牙接收器多以GPS為主，功能較單一。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q1, B-Q8, A-Q19

A-Q15: 為何選擇藍牙連接？有何優缺點？
- A簡: 無線便利、相容舊PDA；但需配對管理、可能斷線與干擾。
- A詳: 藍牙SPP可讓PDA無線接收GPS資料，排除線材干擾車內布局，且對無GNSS的裝置很實用。優點是彈性高、互換容易；缺點包括配對步驟、多一顆電池管理、抗干擾有限。使用時要注意固定與電源。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q2, C-Q1, D-Q2

A-Q16: 導航軟體常見功能有哪些？
- A簡: 目的地搜尋、路徑規劃、語音提示、離線地圖、POI與軌跡。
- A詳: 典型功能含：多模式路徑（最短/最快/避收費）、語音轉向提示、車道指引、POI查詢、離線地圖管理、路徑偏好設定、軌跡記錄與匯出、夜間模式、速度提醒等。使用者可依需求調整，提升實際導引體驗。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q3, C-Q6, D-Q9

A-Q17: 紙本地圖與電子導航的差異與優缺點？
- A簡: 紙圖不耗電、廣域概覽；電子導航即時定位與動態導引。
- A詳: 紙圖優點是全域視野與無需電力；缺點是無法即時定位、更新慢。電子導航可即時顯示位置與路徑、提供語音提示並快速重規劃；但依賴電力與設備，且需定期更新地圖。兩者可互補，長途探索時尤然。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, C-Q8, D-Q4

A-Q18: 什麼是定位精度、HDOP與可靠性？
- A簡: 精度是誤差範圍；HDOP反映幾何分布品質，影響可靠性。
- A詳: 定位精度指實際位置與估計位置的誤差。HDOP（水平幾何精度衰減）衡量可見衛星的幾何排列，數值愈小代表幾何較佳、精度較好。多路徑、遮蔽、單頻等也會影響可靠性。導航軟體可用濾波與匹配提升體感穩定度。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q8, B-Q9, D-Q7

A-Q19: 什麼是 SBAS（WAAS/EGNOS/MSAS 等）？
- A簡: 卫星增强系统，以改正訊息提升定位精度與完整性。
- A詳: SBAS透過地面基準站估測誤差，將改正訊息上行至地球同步衛星，再廣播給接收機。支援SBAS的接收機可應用電離層、衛星時鐘與軌道改正，提升水平精度與完整性。不同區域有不同系統（如WAAS、EGNOS、MSAS）。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q9, A-Q18, C-Q9

A-Q20: 車用環境對 GPS 的影響（擋風玻璃、金屬膜）？
- A簡: 金屬隔熱膜、車頂遮蔽會削弱訊號，影響定位與TTFF。
- A詳: 車內金屬含量高的隔熱膜、車頂、A柱與高樓遮蔽，都可能減弱訊號或造成多路徑，導致定位不穩或首次定位變慢。建議將接收器置於擋風玻璃近窗處，或使用外接天線與防滑墊固定，減少遮蔽與反射。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q5, D-Q1, B-Q18

### Q&A 類別 B: 技術原理類

B-Q1: GPS 如何運作？其定位原理是什麼？
- A簡: 以多顆衛星時間差進行三邊定位，解算位置與時間。
- A詳: 接收機接收多顆衛星的訊號，透過已知衛星位置與訊號傳遞時間差計算距離，形成多球面交會；同時解未知變數（X、Y、Z、時鐘偏移），至少需四顆衛星。配合星曆、曆書與濾波器，持續輸出位置與速度。多星系與多頻可提高可用性與精度。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q1, A-Q14, B-Q8

B-Q2: 藍牙 GPS 與 PDA 如何通訊？
- A簡: 藍牙配對建立SPP虛擬COM，傳NMEA資料供APP解析。
- A詳: 裝置先以藍牙發現與配對，使用PIN驗證後建立SPP通道。系統創建虛擬COM埠（如COM6），GPS持續輸出NMEA字串（4800/9600bps）。導航軟體開啟對應COM，讀取並解析句子（RMC/GGA等），更新位置與時間。連線穩定度受干擾與睡眠管理影響。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q2, C-Q1, D-Q2

B-Q3: 導航路徑規劃演算法如何設計？
- A簡: 以圖論建模道路，採Dijkstra/A*求最短路徑。
- A詳: 道路網建為加權圖，節點為交叉口，邊為路段，權重可為時間、距離或費用。Dijkstra求最短距離；A*用啟發式（如直線距離）加速搜尋。再加入約束（單行、避收費）與偏好（高速/市區），輸出可導航路徑並產生轉向事件。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: A-Q10, D-Q5, C-Q3

B-Q4: 地圖匹配的機制與流程是什麼？
- A簡: 利用GPS點、速度與道路拓撲，推斷最可能所在路段。
- A詳: 演算法先在GPS點附近搜尋候選路段，評估投影距離、方位一致性與拓撲相容，再以隱馬可夫模型或最短路徑決定序列。配合平滑濾波減少跳動，輸出校正後位置。此流程在城市峽谷中尤為關鍵，支撐穩定轉向提示。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: A-Q11, D-Q7, B-Q5

B-Q5: 偏離路徑偵測與重新規劃如何運作？
- A簡: 比較當前匹配路段與路徑容差，超出即觸發重算。
- A詳: 系統持續計算車輛到目標路徑的距離與拓撲一致性；若連續數次超出容差或方向相反，判定偏離。隨即以當前位置為新起點，重跑規劃演算法，並更新轉向序列與ETA。為避免震盪，常加入滯後與穩定參數。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q10, D-Q5, B-Q4

B-Q6: 語音導航與轉向提示如何生成？
- A簡: 由路徑事件轉換為語音樣板，依距離觸發播報。
- A詳: 路徑規劃後產生事件（轉彎、上匝道、環島等），系統根據地圖屬性生成語句樣板（如「前方300公尺右轉」）。利用距離門檻與速度調整播報時機，並在接近與進入前再次提醒。多語言由語音包與TTS提供。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q6, D-Q6, A-Q16

B-Q7: 冷啟、暖啟、熱啟與 TTFF 流程？
- A簡: 依已知星曆/位置多寡決定首次定位時間長短。
- A詳: 冷啟無任何先驗資訊，需下載曆書與星曆，TTFF最長；暖啟保留部分星曆與大致位置；熱啟有精準時鐘與星曆，幾秒內可解算。環境遮蔽與天線影響下載速率。AGPS/SBAS可進一步縮短或穩定TTFF。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q13, D-Q1, C-Q5

B-Q8: 定位誤差來源與抑制機制有哪些？
- A簡: 多路徑、遮蔽、電離層、時鐘偏移等；以濾波、SBAS緩解。
- A詳: 誤差來源含：衛星幾何（HDOP）、電離層/對流層延遲、多路徑反射、硬體噪聲、時鐘偏移。抑制手段包括：多星系多頻接收、SBAS/差分校正、合理天線擺位、卡曼濾波與地圖匹配，提高整體可靠性與體感平滑。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: A-Q18, B-Q18, D-Q7

B-Q9: SBAS/差分如何改善精度？
- A簡: 廣播改正數據修正時鐘與軌道，降低水平誤差。
- A詳: SBAS由地面基站估測大氣與衛星誤差，透過地球同步衛星廣播改正；接收機應用後可顯著降低水平誤差與完整性風險。差分GPS（DGPS/RTK）則利用近距基站提供更精準改正，RTK甚至達到厘米級，車導航多採SBAS等級。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: A-Q19, A-Q18, C-Q9

B-Q10: ETA（預估到達時間）如何計算？
- A簡: 以剩餘距離與分段速度模型推估，動態更新。
- A詳: 系統將路徑分段，根據道路類型、限速、歷史速度或即時路況估計每段行駛時間，累加得到ETA。行進途中依當前速度、停等紅燈與重新規劃結果動態調整。離線環境多用靜態限速與保守係數。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q3, D-Q10, A-Q16

B-Q11: 離線向量圖與點陣圖的地圖架構差異？
- A簡: 向量以幾何與屬性儲存，縮放清晰；點陣以影像瓦片。
- A詳: 向量地圖存道路幾何與屬性，適合搜尋、路徑計算、佔用空間小且縮放品質佳；點陣地圖由瓦片影像組成，渲染快但搜尋與計算能力弱。導航軟體多用向量圖以支援規劃與匹配，輔以點陣底圖做視覺增強。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q9, C-Q4, D-Q4

B-Q12: 方位與航向如何估計？GPS 航向與羅盤差異？
- A簡: GPS航向由移動向量推估；羅盤量測地磁，靜止可用。
- A詳: GPS航向依連續位置變化估算，需有足夠移動速度；磁羅盤測地磁方向，靜止也能量測但受磁干擾。部分裝置融合陀螺儀與加速度計，提高短時穩定。車導航多靠GPS航向，低速或停車時以羅盤/方位推補。

- 學習階段: 核心
- 關聯概念: D-Q7, C-Q5, A-Q7

B-Q13: 更新率與電源管理如何影響體驗？
- A簡: 更新率高反應靈敏但耗電增；需平衡續航與平滑。
- A詳: GPS更新率（1/5/10Hz）決定位置輸出頻度；高更新率改善轉向時機與路口細節，但耗電增加。藍牙與螢幕亦耗電，需調整螢幕亮度、啟用省電策略、外接電源。導航軟體可做抽樣與插值，兼顧平滑與即時性。

- 學習階段: 核心
- 關聯概念: D-Q8, C-Q5, C-Q9

B-Q14: NMEA 資料流與常見語句如何解析？
- A簡: 以逗號分隔的ASCII句子，如GGA、RMC提供關鍵欄位。
- A詳: 典型RMC含UTC時間、狀態、緯度、經度、速度、方位；GGA含定位品質、衛星數、HDOP、高度；語句以校驗碼結尾。軟體透過COM讀入，分割欄位並轉換格式（度分轉小數度），過濾失效（V）狀態，驅動地圖更新。

- 學習階段: 核心

B-Q15: 虛擬 COM 埠與序列資料讀取機制？

B-Q16: 導航軟體的技術架構如何設計？

B-Q17: 位置歷史與軌跡記錄（GPX）如何運作？

B-Q18: 城市峽谷與遮蔽物場景的定位策略？

B-Q19: 外接天線與增益對接收的影響？

- A詳: 外接天線（如有源天線）可提供更佳靈敏度與方向性，放置於金屬車頂外側或靠近前擋玻璃，可降低遮蔽與多路徑。需注意連接器相容性與供電（有源天線需LNA供電）。天線擺位往往比接收器本體更關鍵。

### Q&A 類別 D: 問題解決類（10題）

D-Q1: GPS 一直無法定位怎麼辦？
- A簡: 到空曠處預熱，檢查電源、天線與TTFF，避免遮蔽。
- A詳: 症狀：長時間顯示無定位或衛星少。可能原因：冷啟需時、車內遮蔽物、金屬膜、多路徑、天線不良。解法：移至空曠處開機3-10分鐘；確認電量充足；將裝置靠近擋風玻璃；必要時外接天線；重啟裝置。預防：行前預熱、固定良好擺位。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q13, C-Q5, B-Q7

D-Q2: 藍牙無法配對或頻繁斷線如何處理？
- A簡: 重設配對、固定SPP COM、避免干擾與節電斷線。
- A詳: 症狀：找不到裝置、輸入PIN失敗、連上即斷。原因：配對資料損壞、干擾、SPP未啟用、省電睡眠。解法：刪除配對重建；靠近裝置；關閉多餘藍牙/2.4GHz裝置；在系統鎖定COM；關閉自動休眠。預防：設為受信任裝置，保持電量。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q1, B-Q2, A-Q15

D-Q3: 導航軟體抓不到 GPS 資料（無 COM 連接）怎麼辦？
- A簡: 檢查COM埠與鮑率、關閉占用程式、重啟重設。
- A詳: 症狀：GPS狀態頁無資料或顯示無法開啟埠。原因：COM錯誤、鮑率不符、他程式占用、權限不足。解法：在設定中更換COM與鮑率（4800/9600）；關閉其他GPS軟體；重新配對建立新COM；重啟PDA。預防：固定專用COM，不同軟體避免同時開啟。

- 學習階段: 基礎
- 關聯概念: C-Q2, B-Q15, A-Q12

D-Q4: 地圖位置有偏差或道路不存在如何修正？
- A簡: 更新地圖與索引，檢查座標系統與版本相容。
- A詳: 症狀：道路錯誤、封閉未更新、定位與路網不符。原因：地圖過期、座標系錯誤、索引損壞。解法：更新至最新地圖；重建索引；確認地圖區域與授權；回報圖資廠商。預防：定期更新、出發前檢查關鍵路段、備用紙圖或替代APP。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q4, B-Q11, A-Q17

D-Q5: 路徑規劃很怪、繞路多怎麼改善？
- A簡: 調整偏好、更新地圖、允許重規劃或手動選路。
- A詳: 症狀：明顯多繞路或不走主要幹道。原因：偏好設定（最短/避收費）、地圖權重不準、路段限行變更。解法：切換為最快路徑；更新地圖；允許高速道路；遇即時封路採手動避開。預防：出發前檢查設定與路徑預覽。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q3, B-Q3, A-Q6

D-Q6: 語音導航沒聲音或太小聲怎麼辦？

D-Q7: 行車中定位漂移、跳點的原因與解法？
- A簡: 城市峽谷、多路徑與HDOP差導致；用匹配與擺位改善。
- A詳: 症狀：位置飄移、跨到平行路。原因：高樓反射、多路徑、衛星幾何差、車內遮蔽。解法：改善擺位靠窗、外接天線；降低更新率加平滑；開啟強化地圖匹配；放慢車速於複雜路口。預防：選擇高靈敏接收器、多星系支援。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q4, B-Q8, A-Q18

D-Q8: PDA 電量消耗過快如何優化？
- A簡: 車充供電、降背光、管理藍牙與更新率、關閉背景程式。
- A詳: 症狀：長途導航續航不足。原因：螢幕高亮、藍牙持續連線、高更新率、背景App耗電。解法：使用車充；降低亮度與螢幕逾時；設定GPS更新率合理值；關閉非必要無線與App。預防：定期檢視電池健康與備用電源。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q13, A-Q3, C-Q5

D-Q9: 導航軟體當機或反應慢如何診斷？
- A簡: 清理快取、關閉背景、更新版本與地圖，檢查儲存。
- A詳: 症狀：卡頓、崩潰、畫面延遲。原因：記憶體不足、資料損壞、版本相容性差。解法：清除快取；關閉其他程式；更新APP與地圖；重新安裝並還原設定；檢查儲存卡狀態。預防：保留足夠空間，定期備份。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q16, C-Q10, A-Q16

D-Q10: 時間/時區錯誤導致 ETA 不準怎麼處理？
- A簡: 同步GPS時間，檢查時區與夏令時間設定。
- A詳: 症狀：ETA明顯偏差、軌跡時間錯。原因：系統時區錯誤、夏令時間設定不當、GPS時間未同步。解法：開啟自動以GPS校時；手動選正確時區；關閉錯誤夏令時間。預防：換地區使用前先檢查時區與時間同步。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q10, A-Q13, C-Q3
