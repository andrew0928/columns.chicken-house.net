# IBM ThinkPad X111 ...

## 問題與答案 (FAQ)

### Q&A 類別 A: 概念理解類

Q1: 什麼是 ThinkPad X 系列？
- A簡: ThinkPad X 系列是輕薄商務筆電，重視攜帶、續航、鍵盤與穩定性。
- A詳: ThinkPad X 系列是 IBM（後為 Lenovo）面向行動商務的輕薄型筆電產品線，特色是輕量機身、耐用機構、優秀鍵盤、指點桿（TrackPoint）與長續航。X31、X40 等型號在當年以可靠穩定聞名，常見於差旅、出國念書、外出會議等場景。X 系列通常犧牲部分效能或擴充性，換取便攜與電力管理的優勢，是重行動力使用者的首選。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q2, A-Q11, B-Q1

Q2: ThinkPad X31 與 X40 的定位差異？
- A簡: X31 著重性能與擴充，X40 更輕薄續航長，偏重行動力。
- A詳: X31 使用 2.5 吋硬碟、較完整的擴充，定位偏向性能均衡；X40 採 1.8 吋硬碟與低電壓 Pentium M，機身更輕薄、續航更佳，但磁碟效能與升級彈性相對保守。若常駐辦公且需求 I/O 速度與容量，X31 較適合；若長時間攜帶、出差與校園移動，X40 在重量與電力上更具優勢。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q7, B-Q14, B-Q15

Q3: 什麼是「X111」？
- A簡: X31 一台加 X40 兩台的組合，戲稱 X111 的家用陣容。
- A詳: 文中以「X111」幽默稱呼家中同時擁有一台 X31 與兩台 X40 的情境，象徵多台同系產品併用。此概念延伸到家庭或團隊採購，同平台可共享配件、知識與維運流程，降低學習與管理成本，包含驅動整合、備援機制與資料互通，形成小型「機群」效益。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q12, C-Q10

Q4: 為什麼選擇 X40 而非 X31？
- A簡: 因輕薄與續航，且內建 SD 讀卡機，更符合行動與相機需求。
- A詳: 文中選擇 X40 的關鍵是便攜性與使用情境：兩位使用者需要出國/替代桌機，且相機皆用 SD 卡，X40 內建 SD 讀卡機，提升資料匯入便利；低電壓 CPU 與輕薄機身讓長時間攜帶更舒適。雖 X31 效能與擴充更佳，但整體體驗與設備相容性（SD）使 X40 更合用。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, A-Q7, B-Q14

Q5: 內建 SD 讀卡機的價值是什麼？
- A簡: 省去外接讀卡器，快速匯入相片檔，減少攜帶負擔。
- A詳: 內建 SD 讀卡機可直接讀取相機記憶卡，免外接周邊，降低攜帶物件與遺失風險。對常拍照、需快速整理上傳的使用者，能縮短流程、提升效率。系統層面透過標準儲存類驅動掛載為可移除磁碟，支援自動播放、索引與備份腳本，是行動工作流中的關鍵便利性。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q3, C-Q6, D-Q2

Q6: 低電壓 Pentium M 是什麼？
- A簡: 低電壓 Pentium M 降低功耗與熱量，延長電池續航。
- A詳: 低電壓（LV）Pentium M 是針對筆電省電優化的處理器，具較低核心電壓與 TDP，搭配 SpeedStep 動態調頻調壓，在輕載時降頻省電、重載時提供足夠性能。此設計平衡效能與續航，特別適合輕薄機型如 X40，帶來低溫、長電力與較安靜的風扇聲學表現。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q2, B-Q8, D-Q8

Q7: 1.8 吋硬碟是什麼？與 2.5 吋差異？
- A簡: 1.8 吋硬碟更小更省電，但容量與速度通常較低。
- A詳: 1.8 吋硬碟以更小尺寸與低功耗見長，常用於超輕薄筆電。相較 2.5 吋，其轉速、快取與盤片密度多較保守，因此連續與隨機存取性能偏低，容量上限也較小。優點是重量輕、耗電低、震動慣量小；缺點是升級選擇少與 I/O 瓶頸，對系統整體順暢度有影響。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q4, B-Q15, C-Q5

Q8: Onboard 記憶體與可擴充插槽的差異？
- A簡: Onboard 固焊省空間，插槽模組可彈性升級容量。
- A詳: Onboard 記憶體直接焊在主機板，優點是節省空間、降低接觸問題；缺點是無法更換。插槽式 SO-DIMM 模組可替換與擴充，依平台支援決定上限。X40 內建 256MB，再加裝單條記憶體以擴充整體容量。規劃時需確認單條容量上限、頻率與相容性。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q5, C-Q1, D-Q4

Q9: Combo 光碟機是什麼？功能涵蓋哪些？
- A簡: Combo 為 CD-R/RW 燒錄與 DVD 讀取的外接光碟機。
- A詳: Combo 光碟機通常具備 CD-R/RW 讀寫與 DVD-ROM 讀取功能，不支援 DVD 燒錄。對於無內建光碟的輕薄機，如 X40，可透過 USB/擴充座連接 Combo，用於安裝系統、讀取媒體或備份。選購時留意供電、介面相容與開機支援能力。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q7, B-Q6, C-Q2

Q10: IBM 小黑鼠是什麼？與 TrackPoint 差異？
- A簡: 小黑鼠為外接滑鼠；TrackPoint 為鍵盤內指點桿。
- A詳: 「IBM 小黑鼠」是 ThinkPad 經典外接滑鼠，外觀與手感延續 ThinkPad 設計語言；TrackPoint 則是內建於鍵盤中央的指點桿，無需移動手掌即可精準操控。TrackPoint 適合文字工作與移動有限的環境；外接滑鼠在桌面空間足夠時提供更連續的指向與舒適握持。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q13, D-Q10

Q11: X40 的核心價值是什麼？
- A簡: 輕薄續航、攜帶便利、整體穩定，適合行動工作流。
- A詳: X40 的價值在於便攜與續航平衡：低電壓 CPU、1.8 吋硬碟與輕巧機構設計，帶來低重量與長電池時間；搭配內建 SD 讀卡機，對相片工作流友善。雖 I/O 性能與升級性有限，但對長時間外出、會議筆記、文書與網路應用，提供穩定可靠體驗。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: A-Q4, A-Q7, B-Q8

Q12: 為何家用多台同系筆電有利？
- A簡: 同平台共用配件、驅動與知識，降低維護與學習成本。
- A詳: 多台同系筆電可共享外接光碟、電源供應器、滑鼠等配件；系統安裝流程、驅動與備份策略一致，故障排除經驗可快速複用。家庭環境中，也利於建立統一的檔案同步與備援方案，降低管理複雜度與總體持有成本，提升穩定性與可用性。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: C-Q10, D-Q1

Q13: 出國唸書為何適合輕薄筆電？
- A簡: 輕薄機減輕通勤負擔，長續航應對移動學習場景。
- A詳: 出國學習常需在校園、圖書館、咖啡廳間移動，輕薄與長續航能顯著降低負荷與插電依賴。內建讀卡機等 I/O 便利性讓彈性創作、課業檔案交換更順暢。穩定鍵盤輸入與可靠機構能承受長時間使用，整體提升學習效率與體驗。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, A-Q11

Q14: 如何評估筆電購買價格與議價空間？
- A簡: 先確立需求與市價，捆綁配件與多台採購更具議價力。
- A詳: 評估前先明確需求與同級市價，蒐集多家報價作為談判籌碼。以配件捆綁（外接光碟、滑鼠、記憶體）或多台合購爭取總價折扣與加值服務（升級、保固）。確認通路正規、保固條款、發票與退換規則，兼顧價格與風險控管。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: C-Q1, C-Q2

Q15: 「敗家會傳染」揭示的決策心理是什麼？
- A簡: 近身示範與口碑影響強，降低風險感，促成跟風採購。
- A詳: 家人看到 X31 使用滿意，形成社會認同與近身示範，降低新機風險認知，促成相同品牌採購。這反映口碑、可見度與使用情境匹配度，往往比規格表更能驅動決策。對團體採購，正向體驗可快速擴散，建立一致平台的集體效益。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q3, A-Q12

### Q&A 類別 B: 技術原理類

Q1: X40 的硬體架構由哪些核心組件構成？
- A簡: CPU、晶片組、記憶體、1.8 吋 HDD、I/O（含 SD）、電源與散熱。
- A詳: X40 以低電壓 Pentium M 搭配對應晶片組，含記憶體控制、顯示與 I/O 橋接。儲存採 1.8 吋硬碟，透過 PATA 介面連結南橋。I/O 包含 USB、網路、音訊與讀卡器（SD）。電源子系統管理充放電、供電轉換；散熱模組含熱導管與風扇。整體協同以 BIOS/EC 啟動、OS 驅動接手運作。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q11, B-Q2, B-Q4

Q2: 低電壓 Pentium M 與 SpeedStep 如何運作？
- A簡: 透過動態調頻調壓，依負載在多個 P-State 間切換。
- A詳: SpeedStep 監測負載、溫度與電源狀態，於多個性能狀態（P-States）切換頻率與電壓，降低功耗與熱量。EC/BIOS 提供基礎策略，OS 電源管理（ACPI）與驅動執行決策。輕載時降頻降壓節能，重載時提升時脈保性能。低電壓款具更低標稱電壓，進一步延長續航。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q6, B-Q8, D-Q8

Q3: SD 讀卡機在系統中的運作機制是什麼？
- A簡: 以控制器透過總線連接，由儲存類驅動掛載成磁碟。
- A詳: 內建 SD 讀卡器由控制器連接至南橋（常經由 PCI/USB 內部匯流排）。OS 藉標準儲存類驅動（如 sdbus/sdhost）辨識卡片，掛載為可移除磁碟，支援檔案系統讀寫與熱插拔事件。插入時產生中斷，驅動完成初始化、容量與檔系偵測，再交由檔案系統處理。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q5, C-Q6, D-Q2

Q4: 1.8 吋 PATA 硬碟在 X40 的存取流程如何？
- A簡: 控制器透過 PATA 佇列指令，完成快取、尋道與資料傳輸。
- A詳: OS 發出讀寫要求至儲存堆疊，PATA 控制器以 ATA 指令與硬碟溝通，執行快取命中、磁頭定位、資料傳輸。1.8 吋碟具較低轉速與密度，隨機延遲較明顯，OS 以預取、合併 I/O 與排程降低延遲。電源管理下，碟片可停轉省電，但喚醒時延遲增加。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q7, C-Q5, D-Q1

Q5: X40 記憶體子系統如何運作？
- A簡: 晶片組控制單通道 SO-DIMM 與 Onboard 記憶體協同。
- A詳: X40 晶片組提供單通道記憶體控制器，Onboard 固焊容量與插槽模組共同組成總容量。控制器負責時序、位址映射與錯誤偵測（非 ECC）。單通道對頻寬有上限，I/O 密集時較易受限。最佳化可藉提升總容量、降低分頁與適配時序穩定的模組。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q8, C-Q1, D-Q4

Q6: USB 外接光碟的開機流程為何？
- A簡: BIOS 啟用 USB Legacy，識別裝置並載入外部開機程式。
- A詳: 開機階段 BIOS 初始化硬體並啟用 USB Legacy 模式，掃描啟動順序清單，若偵測到支援的 USB 光碟，讀取其開機紀錄（El Torito）載入開機扇區或引導程式，再轉交控制權。若未啟用或排序靠後，則不會自 USB 開機，需要在 BIOS 調整。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q9, C-Q2, D-Q3

Q7: Combo 光碟機的讀寫技術原理是什麼？
- A簡: 以雷射多功率模式，寫入 CD、讀取 DVD 不同波長。
- A詳: Combo 採單組或雙雷射系統，藉不同波長與功率對應媒體反射特性。CD-R/RW 寫入需校準功率與追蹤伺服，DVD 僅讀取以較短波長提升密度。韌體負責錯誤校正（CIRC、ECC）與緩衝保護。介面多為 ATAPI/USB 封裝，提供光碟抽取與狀態指示。
- 難度: 中級
- 學習階段: 進階
- 關聯概念: A-Q9, C-Q2, D-Q3

Q8: 電源管理如何延長續航？
- A簡: 透過 CPU/磁碟/螢幕省電策略與裝置睡眠協同節能。
- A詳: 系統透過 ACPI 協定與電源計畫控制 CPU P/C-States、關閉未用裝置、調整螢幕亮度與磁碟停轉。網卡、省電驅動與應用層（如背景同步）協同減少喚醒。使用者配置平衡/省電計畫，結合行為（合蓋睡眠、定時關閉顯示），可顯著延長續航。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q6, C-Q4, D-Q6

Q9: 無線網路模組與驅動如何協作？
- A簡: 驅動管理射頻與連線，OS 網路堆疊處理協定與路由。
- A詳: 無線網卡韌體負責射頻控制、加密與連線維護；驅動提供 OS 介面，處理掃描、關聯、功率控制與省電模式。OS 網路堆疊處理 TCP/IP、路由與套接字。省電時可降低發射功率與輪詢頻率；不穩時需調整驅動版本與省電選項。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: D-Q7, C-Q4

Q10: Windows 裝置驅動的安裝流程是什麼？
- A簡: PnP 辨識硬體，匹配 INF 驅動，安裝並載入服務。
- A詳: 開機或插入裝置時，PnP 以硬體 ID 尋找最佳匹配 INF，安裝對應驅動與服務，寫入登錄並載入至核心/使用者模式。簽章驗證、依賴元件（過濾驅動）與相容性檢查決定是否成功。Lenovo 工具可自動化下載與安裝，降低遺漏風險。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q3, D-Q2, D-Q9

Q11: 散熱系統與溫控策略如何運作？
- A簡: 熱導管將熱量至鰭片風扇，溫控曲線動態調速。
- A詳: CPU/GPU 透過導熱介面傳至熱導管，再至散熱鰭片由風扇排出。EC 依感測器溫度控制風扇轉速曲線，OS/BIOS 也會因溫度調整 CPU P/C-States。灰塵與老化導致效率下降時，溫度上升、噪音變大，需要清潔與更換導熱材。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: D-Q8, C-Q9

Q12: 機械硬碟震動與檔案系統有何關聯？
- A簡: 震動增延遲與錯誤率，檔案系統快取與排程可緩解。
- A詳: 機械碟受震動影響，磁頭定位誤差與重試增多，造成延遲。檔案系統與 I/O 排程器透過快取、合併請求與延遲寫入，減少隨機 I/O。行動環境建議避免邊移動邊重 I/O，並啟用寫入快取與預取以平滑性能。
- 難度: 中級
- 學習階段: 進階
- 關聯概念: B-Q4, C-Q5, D-Q5

Q13: 內建與外接輸入設備的事件路徑如何？
- A簡: 裝置驅動傳遞事件至輸入堆疊，供 GUI 與應用消費。
- A詳: TrackPoint/鍵盤與外接滑鼠透過控制器上報中斷，驅動轉換為標準輸入事件，輸入堆疊排序合併，傳至視窗系統與應用程式。指標焦點與加速度曲線在 OS 層配置。衝突多源於驅動過濾層或設定檔，需校準優先權與靈敏度。
- 難度: 中級
- 學習階段: 進階
- 關聯概念: A-Q10, D-Q10

Q14: X31 與 X40 架構差異如何影響性能與續航？
- A簡: X31 磁碟/擴充較強；X40 更省電更輕，續航更佳。
- A詳: X31 的 2.5 吋 HDD 與較高 TDP CPU 提供更高 I/O 與部分計算性能，但重量與耗電增加；X40 的 1.8 吋 HDD 與 LV CPU 降低功耗與熱量，續航與便攜性更強。實務上，文書/網頁差異不大；大量 I/O/解壓縮/安裝時 X31 更快。選擇取決於使用情境與優先權。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: A-Q2, A-Q7, B-Q8

Q15: X40 儲存介面的限制對升級有何影響？
- A簡: 1.8 吋 PATA 與機身空間受限，升級選項較少。
- A詳: 由於採 1.8 吋 PATA 介面與緊湊機構，X40 可用的硬碟/替代方案選擇較少，容量與速度提升有限。方案含更大容量 1.8 吋碟或相容的固態替代，需注意介面、韌體相容與固定機構。評估前先確認規格與 FRU 相容清單。
- 難度: 中級
- 學習階段: 進階
- 關聯概念: A-Q7, C-Q5, D-Q5

### Q&A 類別 C: 實作應用類

Q1: 如何升級 X40 記憶體至平台可支援上限？
- A簡: 確認單條上限與時序，相容 SO-DIMM 安裝後開機驗證。
- A詳: 步驟：1) 查詢機型單條 SO-DIMM 上限與規格（頻率、電壓）。2) 關機拔電池，卸下底部記憶體艙。3) 以 45° 插入新模組並壓下固定。4) 開機進 BIOS/OS 驗證總容量與雙通道狀態（如適用）。Windows 可用 wmic memorychip、Linux 用 dmidecode 檢查。注意：放電防靜電、使用相容顆粒，若不穩定升級 BIOS。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: A-Q8, B-Q5, D-Q4

Q2: 如何以 USB 光碟為 X40 安裝 Windows？
- A簡: 設 BIOS USB 開機，接上外接 Combo，啟動並安裝。
- A詳: 步驟：1) 接上外接 Combo 光碟機。2) 進 BIOS 啟用 USB Legacy，調整啟動順序（USB ODD 優先）。3) 放入安裝光碟，存檔重開機。4) 依精靈分割、格式化、安裝。5) 進 OS 後安裝驅動。注意：若無法辨識，換 USB 連接埠或開機選單（F12），必要時更新 BIOS。最佳實踐：安裝前備份、使用含驅動 Slipstream 版本。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q6, B-Q7, D-Q3

Q3: 如何安裝驅動程式並用 Lenovo 工具自動化？
- A簡: 先安裝晶片組與網卡，再用 System Update 一鍵完成。
- A詳: 步驟：1) 離線備妥晶片組/儲存/網卡驅動。2) 先裝晶片組與儲存驅動，確保 PnP 正常。3) 連網後安裝 Lenovo System Update，自動掃描並安裝推薦驅動與韌體。4) 驗證裝置管理員無驚嘆號。Windows 可用 pnputil /add-driver *.inf /install 批量匯入。注意：優先安裝關鍵驅動、避免跨代驅動混用。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q10, D-Q2, D-Q9

Q4: 如何設定電源計畫以延長續航？
- A簡: 降亮度、限最高處理器狀態、裝置休眠與網卡省電。
- A詳: 步驟：1) 選擇省電計畫。2) 將螢幕亮度降至舒適最低。3) 設定最大處理器狀態 70–85%。4) 磁碟/顯示器閒置關閉時間縮短。5) 無線網卡省電模式設為最高。命令：powercfg /batteryreport、powercfg /a 查可用狀態。注意：平衡效能與體驗，外接電源時自動切換高效能。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q8, D-Q6

Q5: 如何優化 1.8 吋硬碟的效能？
- A簡: 提升記憶體、關閉索引、排程重組與預取優化。
- A詳: 步驟：1) 增加 RAM 降低分頁。2) 關閉不必要的搜尋索引與背景同步。3) 排程磁碟重組（機械碟）。4) 啟用寫入快取。5) 清理啟動程式。Windows：defrag /C /U /V、services.msc 停索引。Linux：mount noatime、tuned。注意：避免重負載時移動設備，定期 S.M.A.R.T. 檢查。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q4, B-Q12, D-Q1

Q6: 如何備份與同步 SD 卡資料？
- A簡: 使用自動化同步腳本，插卡即備份到指定資料夾。
- A詳: 步驟：1) 規劃備份目的地與目錄結構。2) 寫同步腳本。Windows：robocopy E:\DCIM D:\Photos\SD /MIR /FFT /R:2 /W:2；Linux：rsync -av --delete /mnt/sd/DCIM /home/user/Photos/SD。3) 設定自動執行（工作排程/udev）。注意：使用只新增或鏡像模式，避免誤刪；核對時間戳與檔案系統相容性。

- 學習階段: 核心
- 關聯概念: A-Q5, B-Q3, D-Q2

Q7: 如何以 USB 隨身碟建立可開機安裝媒體？
- A簡: 下載 ISO，製作可開機 USB，於 BIOS 設定啟動。
- A詳: 步驟：1) 取得合法 ISO。2) Windows 使用 Rufus（MBR/BIOS 模式），或命令：diskpart→clean→create partition primary→active→format fs=fat32→assign，xcopy ISO 內容至 USB。Linux：dd if=iso of=/dev/sdX bs=4M status=progress sync。3) BIOS 選 USB 優先。注意：選對分割表與啟動模式相容舊機種。

Q8: 如何更新 X40 的 BIOS 與嵌入式控制器？
- A簡: 下載對應版本，接電池與變壓器，依指示刷寫重啟。
- A詳: 步驟：1) 於官網取得機型對應 BIOS/EC（含版本說明）。2) 充足電量並接上 AC。3) 以可開機媒體或 Windows 工具執行更新。4) 按指示勿斷電，完成後重啟與載入預設。注意：確認版本相容、關閉防毒、備份設定。失敗風險高，務必確保電穩與媒體可靠。

Q9: 如何清潔鍵盤與散熱模組以降低溫度？
- A簡: 表面清潔鍵盤，定期除塵鰭片，必要時更換導熱膏。
- A詳: 步驟：1) 關機斷電。2) 鍵盤用微濕無絨布與刷具清潔。3) 拆後蓋以氣吹/刷子清除散熱鰭片與風扇積塵。4) 視需要更換導熱膏與導熱墊。注意：避免高壓氣直吹風扇造成反轉損傷，防靜電；拆機依維修手冊操作，保持螺絲對位。

Q10: 如何建立家庭共享與多台筆電資料同步？

- A詳: 步驟：1) 指定一台為共享主機（或 NAS）。2) 啟用 SMB 共享與權限；Windows：net share Docs=D:\Docs /grant:Users,CHANGE。3) 用同步工具（Syncthing、rsync、OneDrive）維持一致。4) 設定排程與衝突解決策略。注意：備份三二一原則、區分私人與共用、加密重要資料。

### Q&A 類別 D: 問題解決類

Q1: X40 開機或操作緩慢怎麼辦？
- A簡: 檢查硬碟健康、升級記憶體、精簡啟動與磁碟優化。
- A詳: 症狀：開機久、程式啟動慢、磁碟燈長亮。原因：1.8 吋 HDD I/O 限制、RAM 不足、背景程式過多、磁碟碎片或壞軌。解法：1) 檢查 S.M.A.R.T.；2) 增 RAM；3) 關閉索引與自動啟動；4) 重組磁碟或更換更快儲存。預防：定期維護、避免重負載下移動設備、良好備份。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q4, C-Q5

Q2: SD 讀卡機不辨識如何處理？
- A簡: 更新驅動、檢查裝置管理員與更換 SD 測試。
- A詳: 症狀：插卡無反應或裝置錯誤。原因：驅動缺失/損壞、接點氧化、SD 格式或容量不相容。解法：1) 重新安裝或更新驅動；2) 用酒精清潔卡金手指；3) 測試其他卡或 USB 讀卡器；4) 檢查磁碟管理是否未指派代號。預防：安裝官方驅動、避免熱插拔拔出前未卸除。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q3, C-Q6

Q3: USB 光碟無法開機如何診斷？
- A簡: 確認 BIOS 設定、媒體可啟動性與硬體連接。
- A詳: 症狀：無法從光碟進入安裝。原因：BIOS 未開 USB Legacy/啟動序錯誤、光碟非可開機、USB 埠供電不足。解法：1) 啟用 USB 開機並調整順序；2) 重燒可開機媒體；3) 改用另一 USB 埠或加供電；4) 更新 BIOS。預防：安裝前驗證媒體、隨身攜帶備援 USB 啟動碟。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q6, C-Q2, C-Q7

Q4: 升級記憶體後不開機怎麼辦？
- A簡: 檢查相容性與安裝，清除 CMOS，單條逐一測試。
- A詳: 症狀：黑屏、蜂鳴或重啟循環。原因：容量/時序不相容、接觸不良、BIOS 舊。解法：1) 重新插拔，擦拭金手指；2) 單條測試定位問題模組；3) 更新 BIOS；4) 更換相容規格。預防：查相容清單、避免超規格模組、帶防靜電操作。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q5, C-Q1

Q5: 硬碟異音或壞軌如何處置？
- A簡: 立即備份，檢查 S.M.A.R.T.，規劃更換與資料救援。
- A詳: 症狀：咔嗒聲、讀寫錯誤、系統凍結。原因：機械磨損、撞擊、壞軌擴散。解法：1) 立刻備份關鍵資料；2) 用工具檢測（CrystalDiskInfo、smartctl）；3) 規劃更換相容儲存；4) 必要時交由專業救援。預防：避免震動、定期備份、監控 S.M.A.R.T.、保持通風。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q4, B-Q12, C-Q5

Q6: 電池續航變短的原因與對策？
- A簡: 容量老化、設定不佳；校正、優化電源與更換電池。
- A詳: 症狀：滿電使用時間明顯縮短。原因：循環老化、溫度過高、背光偏高、常滿充。解法：1) 校正電量計（完整充放一次）；2) 調整電源計畫與亮度；3) 更新 BIOS/EC；4) 老化嚴重時更換原廠或相容電池。預防：避免長期高溫滿充、定期健康檢查。

- 關聯概念: B-Q8, C-Q4

Q7: 無線網路斷線或不穩如何修復？
- A簡: 更新驅動與韌體，關閉過度省電，調整頻道與加密。
- A詳: 症狀：頻繁斷線、速率波動。原因：驅動相容性、AP 設定、干擾、省電策略。解法：1) 更新網卡驅動與 AP 韌體；2) 關閉網卡省電；3) 固定 AP 頻道、避開干擾；4) 重設網路設定與清除配置。預防：穩定加密（WPA2）、定期更新、最佳化 AP 位置。

Q8: 溫度過高或風扇噪音大如何處理？
- A簡: 清潔散熱、換導熱材，調整風扇曲線與電源策略。
- A詳: 症狀：高溫降頻、風扇全速。原因：積塵、導熱老化、軟體負載、環境高溫。解法：1) 清潔鰭片與風扇；2) 更換導熱膏；3) 檢查背景程序；4) 調整電源計畫限制最大時脈。預防：定期保養、良好通風、避免軟體長期占用。

Q9: 系統更新後出現藍畫面怎麼辦？
- A簡: 進安全模式回退驅動或更新，檢查事件與記憶體。
- A詳: 症狀：更新後藍屏重啟。原因：驅動衝突、韌體不相容、記憶體錯誤。解法：1) 安全模式卸載近期驅動/回復點；2) 更新至官方建議版本；3) 檢查事件檢視器/小型傾印；4) 執行記憶體測試。預防：分階段更新、建立映像備份、嚴選驅動來源。

- 關聯概念: B-Q10, C-Q3, C-Q8

Q10: 外接滑鼠或 TrackPoint 失靈如何診斷？

- A詳: 症狀：指標不動、飄移或間歇性失效。原因：USB 接觸不良、驅動過濾層衝突、電源管理關閉裝置。解法：1) 更換 USB 埠/線材；2) 重新安裝或回退驅動；3) 關閉輸入裝置省電；4) 檢查系統設定與惡意軟體。預防：維持驅動穩定版本、定期清潔、禁用不必要的過濾驅動。
