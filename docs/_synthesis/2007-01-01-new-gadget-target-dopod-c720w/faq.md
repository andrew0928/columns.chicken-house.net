# 敗家新目標... Dopod C720w（智慧型手機選購與比較 FAQ）

## 問題與答案 (FAQ)

### Q&A 類別 A: 概念理解類

Q1: 什麼是智慧型手機（SmartPhone）？
- A簡: 可安裝應用、具行動上網與PIM同步的行動裝置，比功能型手機更像袖珍電腦。
- A詳: 智慧型手機是可執行多種應用程式、連網傳輸資料，並能與行事曆、連絡人、郵件等個人資訊管理（PIM）服務同步的行動裝置。以文中環境為例，Windows Mobile Smartphone 搭配 Outlook/Exchange 可整合郵件、行事曆、工作及聯絡人，並支援 2G/3G 或 Wi‑Fi 等連線。與功能型手機相比，智慧機更重視擴充性、生產力與連線能力，但也更仰賴電力與系統穩定度。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q4, B-Q1, A-Q11

Q2: 什麼是待機時間？與通話時間有何差異？
- A簡: 待機時間是不操作時可持續開機的時長；通話時間是持續通話的可用時長。
- A詳: 待機時間指手機在無大量操作、螢幕關閉、僅維持網路待命與後台同步等情況下，從充飽電到耗盡電的可用時長；通話時間則是在持續通話狀態下，基地台連線與音訊編碼長時間運作時的可用時長。兩者受電池容量、基帶耗電、網路訊號強度、同步設定、背光與無線模組（3G/Wi‑Fi/藍牙）是否開啟等因素影響。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: D-Q1, B-Q8

Q3: 為何低電量時更容易通話中斷？
- A簡: 低電壓下基帶功率受限，訊號餘裕不足，易觸發掉話或主動斷線保護。
- A詳: 當電量過低，電源管理晶片為避免過度放電，會限制瞬時輸出電流與無線電發射功率。若通話中遇到訊號衰減，手機需提高發射功率維持連線，但受限於低電壓可能無法達到所需功率，造成訊雜比不足而掉話。部分裝置會在臨界電壓主動關閉射頻以自保，表現為電量掉格後立即斷訊。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: D-Q2, B-Q3

Q4: 為什麼需要智慧型手機搭配 Outlook 同步？
- A簡: 無縫同步郵件、行事曆與聯絡人，行動中維持資訊一致與工作效率。
- A詳: 智慧型手機結合 Outlook（或 Exchange/Office 365）可把郵件、行事曆、聯絡人與工作待辦雙向同步，減少手動匯入匯出或重複管理的成本。對經常移動的使用者，這意味著約會提醒、最新郵件與聯絡資訊隨時更新，回到電腦也自動一致。搭配 DirectPush 還能接近即時收信，但須兼顧耗電與行動資料成本的取捨。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q1, B-Q15, C-Q1

Q5: 3G 與 Wi‑Fi 有何差異？該如何取捨？
- A簡: 3G覆蓋廣行動性佳、費率較高；Wi‑Fi速度快成本低，但依賴熱點可用性。
- A詳: 3G（如 WCDMA/HSPA）提供廣泛戶外覆蓋、不中斷的移動連線，適合隨時隨地使用；但資費通常較高，延遲與耗電也相對大。Wi‑Fi（802.11b/g）在室內或熱點下速度快、延遲低且無量計費常見，但受限於熱點可用性與安全設定。取捨視個人使用情境：若多在辦公室或家中，Wi‑Fi 足矣；若長時間通勤或外勤，3G 便利性更高。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q2, B-Q6, B-Q7, C-Q3, C-Q2

Q6: 什麼是 QVGA 與 WQVGA（如 240×320 與 360×240/320×240）解析度？
- A簡: QVGA 為 240×320 標準直向；WQVGA/橫向寬螢幕常見 320×240 或近似寬比例。
- A詳: QVGA 指 240×320 像素的螢幕，軟體支援最普遍；WQVGA 泛指較寬的低階解析度，如 320×240 橫向或 360×240 等變體，顯示橫向資訊較多，瀏覽郵件與網頁更舒適。然而，一些軟體若寫死為直向 QVGA 或未支援旋轉/寬比例，可能出現 UI 錯位或裁切，故需注意相容性。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q4, D-Q5

Q7: 橫向寬螢幕的核心價值與限制是什麼？
- A簡: 強化橫向資訊密度與閱讀體驗，但可能遇到介面未適配的相容性問題。
- A詳: 橫向寬螢幕在同等像素下能橫向顯示更多字元，對郵件、簡報預覽、網頁與即時通訊對話尤為有利；配合實體鍵盤，輸入與閱讀可同時兼顧。限制在於部分應用採用固定版面或未支援橫向，導致排版錯位或功能鍵出界；此外，較寬螢幕在單手操作的觸及性與捲動距離上也需適應。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q6, D-Q5

Q8: 實體 QWERTY 鍵盤與數字鍵（T9）輸入有何差異？
- A簡: QWERTY 有實體反饋與快捷鍵，長文輸入效率高；T9 輕巧但長文易疲乏。
- A詳: QWERTY 鍵盤提供清楚的鍵程與多鍵快捷（如郵件、日曆熱鍵），適合大量文字輸入、郵件回覆與即時通訊；成本是機身寬一些、單手操作需適應。數字鍵配合 T9 智能輸入能維持輕薄外型，短訊與撥號快速，但長文輸入與指令操作效率較低。選擇應依日常輸入量與使用習慣決定。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: C-Q8

Q9: 為何「軟體相容性」在選機時很重要？
- A簡: 不同解析度與方向、系統版本差異，會影響應用是否正常顯示與運作。
- A詳: 相容性包含螢幕解析度/方向（QVGA vs WQVGA/橫向）、系統核心版本、API 可用性與輸入裝置支援。若應用介面寫死或未適配，可能出現畫面裁切、按鈕不可見或閃退；驅動與編碼差異也會影響多媒體與通訊功能。以文中取捨為例，選擇多用內建軟體可降低第三方相容風險。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q10, D-Q5

Q10: 為什麼有人會放棄 3G，改以 Wi‑Fi 與內建軟體為主？
- A簡: 常在室內網路足、成本考量與續航需求下，3G 價值不高而選擇 Wi‑Fi。
- A詳: 若使用情境以家中/公司為主，Wi‑Fi 涵蓋與速度已能滿足收信與同步；加上 3G 資費較高、耗電較大，對不常外出使用資料者，性價比不佳。搭配內建郵件/行事曆等應用，降低第三方相容性風險，讓體驗更穩定。此取捨也常見於有成本與續航優先的使用者。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, C-Q2, C-Q3

Q11: 選購智慧型手機的核心指標有哪些？
- A簡: 通訊可靠、電力續航、連線能力、螢幕與輸入、軟體生態與價格。
- A詳: 核心指標包括：1) 通訊可靠度（訊號、掉話率、通話穩定）與語音品質；2) 電池容量與待機/通話時間；3) 連線能力（3G、Wi‑Fi、藍牙）及費率；4) 螢幕尺寸/解析度與可讀性；5) 輸入方式（QWERTY、觸控、T9）；6) 軟體相容性與生態；7) 價格與保固服務。可依自身使用情境設定權重進行比較。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: B-Q12, C-Q9

Q12: 同價位產品要如何做取捨？
- A簡: 以需求加權評分，將功能對應使用情境，量化比較後決策。
- A詳: 當價格相當時，建議列出需求（如通話可靠、電力、鍵盤、Wi‑Fi/3G、相容性）並賦予權重，再對每台裝置逐項打分，計算加權總分。這能避免被單一賣點影響，確保選擇真正符合日常情境。若重要指標拉鋸，可回到關鍵任務（如長時間通話/大量收發信）進行實機測試驗證。
- 難度: 初級
- 學習階段: 核心
- 關聯概念: A-Q11, C-Q9

Q13: 什麼是門號綁約？對換機時程的影響？
- A簡: 綁約是以優惠換取使用承諾，未到期解約需付違約金或回補補貼。
- A詳: 門號綁約屬合約義務，使用者以承諾一定期間的月租與資費換取手機補貼或費率優惠。未到期更換門號或解約通常需支付違約金或補貼差額。實務上可等待期滿、以空機入手或評估違約成本。綁約不同於 SIM 網路鎖，後者是技術限制裝置只能用特定電信商的 SIM 卡。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: D-Q10, B-Q14

Q14: Dopod C720w 與 595 的主要差異是什麼？
- A簡: C720w 有實體鍵盤、橫向寬螢幕與 Wi‑Fi；595 具 3G、標準 QVGA，軟體更相容。
- A詳: 依文中描述：C720w 配備 QWERTY 鍵盤、橫向寬螢幕（如 320×240/360×240）、內建 Wi‑Fi，強調文字輸入與室內高速上網；595 支援 3G 行動數據、採 240×320 標準 QVGA，第三方軟體相容性更佳。兩者官方定價相當，取捨在於室內 Wi‑Fi 與輸入效率，或是戶外 3G 便利與軟體生態。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q5, A-Q6, C-Q2, C-Q3

Q15: 為何偏好使用內建軟體可降低相容風險？
- A簡: 內建軟體隨系統與硬體共同設計，較能適配解析度、方向與輸入裝置。
- A詳: 內建應用由系統廠商針對該機型與作業系統版本最佳化，能正確處理螢幕解析度/方向、字型與輸入法；同時在能源管理、背景同步與連線管理上更穩定。第三方應用若未充分適配特定解析度（如寬螢幕）或舊版 API，容易出現顯示與效能問題。以內建為主能降低不確定性，維持一致體驗。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: A-Q9, D-Q5

### Q&A 類別 B: 技術原理類

Q1: Windows Mobile 與 Outlook 的同步機制如何運作？
- A簡: 透過 ActiveSync/WMDC 建立夥伴關係，同步郵件、行事曆、聯絡人與工作。
- A詳: 同步由 ActiveSync（XP）或 WMDC（Vista 以後）在 PC 上建立與手機的夥伴關係。透過 USB/藍牙或 Exchange OTA，系統比對 PIM 資料庫（郵件、日曆、聯絡人、工作），依衝突策略合併。核心元件包含：PC 端同步服務、手機 PIM 資料庫、連線層與（可選）Exchange Server。流程為配對、選擇項目、第一次全量同步、之後僅傳遞差異。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q1, A-Q4

Q2: 手機如何在 3G 與 Wi‑Fi 間選擇連線？（Connection Manager）
- A簡: 由連線管理器依優先序與規則選路，Wi‑Fi 可設為優先，無可用時回退行動網。
- A詳: Windows Mobile 的 Connection Manager 維護多組連線設定檔（Wi‑Fi、GPRS/3G、代理）。當應用請求網路時，系統依優先序選擇可用介面，Wi‑Fi 一旦連上通常優先於行動數據；若 Wi‑Fi 中斷則自動回退 2G/3G。核心組件含介面偵測器、路由策略、APN/SSID 設定與憑證儲存，確保連線順暢與最小中斷。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q2, C-Q3, A-Q5

Q3: 電源管理與電池保護如何影響通話穩定？
- A簡: 低壓保護與電流限制會壓縮無線發射功率，導致低電量下易掉話。
- A詳: 電源管理 IC 監控電池電壓與溫度，於低電壓/高負載時限制峰值電流，避免過放與過熱。通話時基帶需動態調整發射功率以對抗路徑損耗；當供電能力不足，將無法維持必要的 EIRP，進而造成封包重傳、語音失真與掉話。系統亦可能在臨界值觸發射頻關閉，保護電池壽命與資料完整性。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: D-Q2, A-Q3

Q4: 解析度與 DPI 縮放對應用介面有何影響？
- A簡: 未適配寬螢幕/旋轉的程式會因像素密度與版面固定而產生裁切或錯位。
- A詳: Windows Mobile 的 GDI 以邏輯單位繪製，透過 DPI 與方向參數轉為實像素。若應用硬編 QVGA 直向座標或使用位圖定尺寸，旋轉到橫向或 WQVGA 時會溢出與裁切。支援的程式會偵測方向變化、使用自動佈局或資源組態。核心組件含顯示驅動、GDI、資源管理與方向通知事件。
- 難度: 高級
- 學習階段: 進階
- 關聯概念: A-Q6, D-Q5

Q5: 實體鍵盤輸入事件在系統中如何處理？
- A簡: 鍵盤掃描碼經鍵盤驅動與輸入法處理，轉為字元/命令供應用接收。
- A詳: 按鍵觸發掃描碼由鍵盤控制器傳至驅動，映射為虛擬鍵（VK_）後交給 IME/輸入法組件產生字元或命令。系統支援快捷鍵映射與長按/組合鍵。應用程式透過訊息循環接收鍵盤事件。核心組件：鍵盤驅動、IME、訊息迴圈、快捷鍵設定。硬鍵可綁定系統功能或應用啟動以提升效率。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: C-Q8, A-Q8

Q6: 3G 技術特性如何影響體驗與耗電？
- A簡: 3G 帶來更高吞吐與移動性，但連線維持與高發射功率使耗電提升。
- A詳: WCDMA/HSPA 透過更寬頻寬與先進編碼提升吞吐與穩定性，支援更佳的移動連續性。RRC 狀態（CELL_DCH/FACH/PCH）間切換與小區搜尋會消耗能量，訊號差時上行功率提升亦加重耗電。優點是隨時可用的資料連線與更低延遲；缺點是對電池、資費較敏感。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q5, C-Q3

Q7: Wi‑Fi 的連線原理與功耗特性是什麼？
- A簡: 透過 AP 關聯與節能輪詢傳輸資料，吞吐高延遲低，但掃描與弱訊號會耗電。
- A詳: 手機掃描 SSID、選擇信道，與 AP 完成認證（WEP/WPA/WPA2）與關聯，之後以 802.11 MAC 傳輸。省電模式會在訊框（Beacon）間睡眠，降低耗電；但頻繁掃描、弱訊號重傳或高吞吐下功耗增加。室內環境干擾（同信道/重疊信道）亦影響穩定性與耗電。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: A-Q5, C-Q2, D-Q3

Q8: 待機耗電由哪些部分組成？如何估算？
- A簡: 基帶待命、週期性同步、感測/背景程序與硬體維持電流共同決定。
- A詳: 待機耗電包含：基帶 DRX 週期聽呼、週期性保活（如 Push/IM）、排程同步、時鐘/記憶體維持電流，以及偶發喚醒（通知、掃描）。估算可用電池容量（mAh）除以平均待機電流（mA），並考量高耗時段（背景同步）。降低背光、關 3G 或縮短同步頻率可顯著延長待機。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: D-Q1, C-Q4

Q9: 通話掉話如何被偵測與處理？
- A簡: 由網路與終端偵測無線連結失效，發出掉話事件並嘗試重選/重撥。

Q9: 手機發熱且耗電異常快，如何診斷？
- A簡: 檢查背景同步、網路搜尋與訊號強度，必要時重置或換電池。
- A詳: 症狀：待機也發熱、電量快速下降。原因：Push 頻繁喚醒、弱訊號導致高功率、Wi‑Fi 掃描、程式迴圈。解法：1) 暫停 Push/改定時；2) 關閉多餘無線；3) 觀察訊號差時改 GSM；4) 重新開機與檢視可疑程式；5) 換新電池。預防：良好網路環境、精簡常駐。
- 難度: 中級
- 學習階段: 核心
- 關聯概念: B-Q8, C-Q4

Q10: 綁約未到期但想換機怎麼辦？
- A簡: 評估違約金、改買空機或等待期滿，或暫以 Wi‑Fi 使用資料。
- A詳: 症狀：綁約限制導致換機受阻。方案：1) 詢問違約金與回補補貼成本；2) 直接購買空機換用原門號 SIM；3) 若要新方案，先等期滿或攜碼；4) 不需更動門號亦可先入手裝置，用 Wi‑Fi/2G 依需求使用。預防：簽約前評估裝置汰換週期與需求變化。
- 難度: 初級
- 學習階段: 基礎
- 關聯概念: B-Q14, A-Q13
