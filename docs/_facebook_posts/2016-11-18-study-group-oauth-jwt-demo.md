今天讀書會的錄影 :D

Demo 的部分我用 Azure API App service 的專案類型來示範，搭配 swagger 來 demo API 很方便 (Y)
這次是用 POC 的方式，來展示怎麼做到像 OAuth, JWT 這些功能。POC 的好處是程式碼很短，百來行的 code 就能表達出這樣的架構，用來研究是個不錯的選擇。做過 POC 你對這些技術的了解就更上一層樓了，用起來也會更得心應手，不會被技術還有規格牽著跑。

既然都已經用 azure api app service, 我就直接 放在 azure 上面給大家試用了, 網址請直接參考 GitHub 的說明。
