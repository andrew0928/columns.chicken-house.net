每年 7/1 都要緊張一次，等著 MVP 連任開獎，今年又幸運過關一次 :D

掛著 MVP 的名號，其實除了 Microsoft 給的一些授權，以及一些 Azure 的額度之外，其實就跟一般人沒兩樣了。MVP 鼓勵的是社群的分享，過去一年工作繁忙，除了寫了幾篇文章，以及講了兩場演講 ( DevOpsDays / .NET Conf ) 之外，就沒有其他可以拿出來說嘴的貢獻了。

還好這次評審賞光，給了我繼續連任的機會，特地貼張照片紀念一下。

其實 MVP 的身分只是個 Microsoft 對於社群貢獻肯定的名份而已，即使沒有這 Logo, 我也是會持續寫文章跟分享的... 我的部落格都寫了快 20 年了，MVP 的圈圈到今年也才拿到第五個而已 XDD, 未來一年我會持續努力, 也請各位繼續支持 :D
