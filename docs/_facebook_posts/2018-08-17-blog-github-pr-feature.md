部落格上添加了一個小功能 (雖然也沒啥人會去用)..

我的部落格已經遷移到 GitHub Pages 好一陣子了，其實只要把文章 push 上去，過個一分鐘就會 publish 了。平常寫文章就像寫 code 一樣的 process...

既然東西都在 github 上, 對我文章內容有任何建議我也都很歡迎。如何在 github 上貢獻? 最簡單的方式就是 pull request 了。

沒錯，我添加的功能，只是添加個正確的 github link 而已，每篇文章右側有個 "Edit Page" 連結，點下去就能編輯提出 pull request 了

小到錯字或是勘誤，大到內容補充都歡迎 :)
