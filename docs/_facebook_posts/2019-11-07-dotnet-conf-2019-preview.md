大家後天見, 有來參加 .net conf 2019 的歡迎來聽我的場子 :)

這次挑戰連續講兩場, 第一場講架構的設計與規劃, 內容跟上個月 DevOpsDays Taipei 講的 "從零開始的 DevOps" 其實是同一件事，不過在那場沒辦法講 .NET 啊，很多內容都只能點到為止...

這次終於等到 .NET Core / C# 的主場了, 我多講一點 C# 也不會有人白眼, 就決定一次講兩場把它講完.. :D

我會從整個團隊要轉進微服務的架構規劃開始 (#1)，到架構團隊 & 基礎建設團隊應該先做好什麼樣的準備 & 開發框架，來讓開發團隊能用最低的進入門檻轉進微服務... 

第二場會有很多的 code & demo, 為了這次 demo 我特地把 webapi 用 .NET Core 3.0 / Generic Host 改寫了一次.. 有興趣的記得來參加 :D

這次大會的日子實在太會挑，竟然挑在雙十一的前一天.. Orz, DevOpsDays 結束不到一個月, 又碰上雙十一正忙, 還要準備兩場 session... 這次就算爆肝也要好好的準備... 希望講的內容大家都會喜歡。這次我們也會有攤位，有空歡迎來攤位找我聊聊~, 大家後天見 :)
