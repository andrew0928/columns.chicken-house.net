因為疫情的關係，好一陣子沒有參加這類社群活動了。這次剛好 MVP 同梯 Shengyou Fan 的邀約，就答應參加這次 DevOps 主題的聊天了..

時間是 10/20 20:00 ~ 22:00, 免費報名, 有興趣的快把握機會。這次主題圍繞在 DevOps 身上，其實我跟另外幾位 speaker 在彩排時都在聊，同一個 DevOps 其實有各種解讀啊! 狹義的 DevOps 就是指 CI / CD 的技巧，廣義的則是指 Dev 跟 Ops 的良性循環。再廣義一點的也把敏捷的精神納進來了，持續交付與改善，加速價值的交付。

這次一起出場的幾位都有豐富的經驗，商用軟體開發，開發工具，社群等等都有，涵蓋範圍其實蠻豐富的。我會趁這機會聊聊我們自己團隊運作如何看待 DevOps 的。雖然 run 起來還是覺得有不少地方可以改善，不過既然認真投入心力了，成功或是失敗的經驗都很有分享的價值啊。成功不一定能複製，但是失敗一定會學到教訓，這些都是很好的分享題材。

還剩三天，有興趣參加的記得去報名! 報名時可以附上問題，我們會盡量當場回應。沒回答到的話，也歡迎在這裡留言，我再一一回覆~
