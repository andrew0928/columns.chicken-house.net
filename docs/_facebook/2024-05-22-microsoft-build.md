好久沒有看 live 的 //Build 那麼 high 了 :D

果然 windows 全部都押在 AI 跟 Copilot 身上了。半年前寫的那篇文章，還有試做的 安德魯小舖GPTs, 果然現在都即將在 Copilot+ PC 上面能實現了

雖然這一切都還沒到來 ( AI PC 還買不到，截圖的這些服務都還沒出現 )，但是 Microsoft 預告了有這些 AI 基礎建設會內建在 windows 身上, 開發人員至少有個標準環境可以參考了..

(如果我寫了安德魯小舖的 APP，安裝手冊卻還要引導你下載安裝完整套 ollama 的話，然後還要幫你檢測你的電腦跑不跑得動..)

看了一下，有本地端的 SLM，Text Embedding Model，有 Copilot Stack，有 Microsoft Copilot (以後的 Portal?)，規格還未明，但是至少是個 AI Ready 的環境了。開發者要的不多，這是第一步..

這一切還沒到，不過就快了。
