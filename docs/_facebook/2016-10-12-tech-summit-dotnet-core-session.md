有時候外國的月亮真的比較圓啊 XD

今天親眼看到 Microsoft 的人親自說明跟 demo .NET core, 就算是很簡單的 demo, 也是可以察覺一些我以前不曉得的細節.. 

demo 的都是很基本的東西，不過 speaker 果然是專家，可以把基本的東西背後的 connection 都交代的很清楚，從 .netcore, nuget, console app, library, .net standard, webapp, webhost... 最後還追加 10 min demo docker... (Y)

這些資訊 PPT 上面沒有，平常的 session 也聽不到這些內容... 如果不是夠瞭解這些底層架構的人，大概沒辦法這麼輕鬆的談完這一連串的內容吧! 這次有聽到幾場關鍵的 session 就算值回票價了 (Y)

老外講技術的方式，還有 demo 的方法真的值得學一下，聽它們講起來就真的比較厲害... 被我一講就只剩 80% .. 重新練習一下，之前那幾篇文章看來要找時間重寫一次了..
