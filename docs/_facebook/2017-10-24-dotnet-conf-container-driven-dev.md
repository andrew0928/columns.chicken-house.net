2017/10/14 在台中逢甲大學，由 #Study4TW 主辦的 .NET Conf 2017, 我主講的場次 "Container Driven Development" 錄影檔出來了, 為了能讓大家看清楚投影片的內容，除了當天 DV 拍攝之外，我自己 NB 也開啟錄影，兩分剪再一起，希望效果會好一點。

內容主要是告訴大家，為何要用容器來部署? 更直接一點，有了容器之後，你的開發方式可以有那些改變? 有哪些 code 是多餘的可以不用再寫? 這些都是我這個 session 想要傳達的。只是可惜時間有限，大部分的 demo 都省掉了，結果還是超時.. XD

最後感謝當天來聽我場次的朋友們，生平第一次得到滿分的成就 :D
