重貼一次上個月的文章，上次貼完後有個遺憾，就是沒有測過直接在 WSL 上面 mount 實體磁碟機的效能.. 感覺這篇就是缺了一塊，直到這週末我補上了一直懸在那邊的疑惑..

效能有改善是預期的，畢竟少了層層轉換；但是意料之外的是有些組合下，SSD 交給 WSL 使用的表現，還比原生 windows 強 (這點我還真猜不透原因，知道的人留言給我點線索 XDD)，除了 Microsoft 還撐著不願意給每次啟動 windows 自動 mount 的功能之外，一切就完美了 XD

這次竄出來的黑馬，是我六年前買的 SSD: 970pro, 大概是末代的消費級 MLC SSD, 我把它直接 mount 到 linux /dev/sdc , 竟然可以跑出 比 windows 還要好上 20% 的效能, 而另一顆骨董的 Intel SSD730 (對，就是 10 年前的) 也跑出不錯的分數，雖然數字差了一些，但是表現也比現在的 QLC 還強 ( 現在的 QLC 還真的是能閃則閃 ) ...

補上一段我在我自己 FB 貼過的結論: 6 年前的 MLC SSD 拚贏現在的 TLC SSD, 而 10 年前的 MLC SSD 也拚贏現在的 QLC SSD .. SSD 還真的是個越發展越走回頭路的領域.. 小容量但是品質好的 SSD ( ex: 256G, or 512G ) 拿來工作用途剛剛好啊。

需要拿 WSL 當作你的生產力環境的人，可以認真考慮一下這投資了。我補了一段測試，放在 2-6, (文章連結見第一則留言) 已經看過我這篇文章的可以直接跳到那段。
